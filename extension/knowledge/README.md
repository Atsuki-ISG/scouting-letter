# ナレッジ参照

スカウト文生成に必要なナレッジは、本リポジトリのルートにあるファイルを参照してください:

- `knowledge/basics.md` — 基本ルール
- `knowledge/structure.md` — 文章構成
- `knowledge/personalization.md` — パーソナライズのコツ
- `knowledge/learnings.md` — 過去の修正から得た共通ルール
- `companies/[会社名]/templates.md` — 会社別テンプレート
- `companies/[会社名]/profile.md` — 会社プロファイル

バッチ生成用プロンプトは `extension/prompts/` 配下に自己完結型として格納されています。
