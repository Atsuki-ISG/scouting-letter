# スカウト文の基本ルール

## スカウト文の目的

- 求職者に「この会社で働きたい」と思ってもらうこと
- 返信・応募へのアクションを促すこと
- 会社と求職者のマッチング精度を高めること

## 心構え

- 一方的なアピールではなく「対話の始まり」を意識する
- 求職者のキャリアや希望を尊重した姿勢を示す
- 誠実さと熱意のバランスを保つ

## 返信率を上げるポイント

### 効果的な要素

- 求職者の経歴・スキルへの具体的な言及
- なぜその人にスカウトを送ったかの理由
- 会社の魅力を簡潔に伝える
- 次のアクションを明確に提示

### 避けるべきこと

- テンプレート感が強い文章
- 長すぎる自社アピール
- 曖昧な表現や抽象的な言葉
- 上から目線の表現

## NGワード・避けるべき表現

### NG表現の例

- 「誰でもOK」「未経験大歓迎」（軽視されている印象）
- 「アットホームな職場」（具体性がない）
- 「やる気次第で」（条件が不明確）
- 「すぐにでも」（焦らせる印象）

### 言い換え例

| NG | OK |
|----|----|
| アットホーム | チームで声を掛け合いながら働いています |
| 未経験OK | 入職後3ヶ月の研修プログラムがあります |
| やる気次第 | 資格取得支援制度で年間○名がキャリアアップ |
