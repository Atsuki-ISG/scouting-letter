# 文章構成のポイント

## 件名の書き方

### 基本原則

- 30文字以内を目安に
- 求職者の名前や資格を入れる
- 興味を引くが誇張しない

### 件名パターン

1. **パーソナライズ型**
   - 「〇〇様の介護福祉士としてのご経験を拝見しました」
   - 「〇〇様へ、訪問介護の経験を活かせる職場のご案内」

2. **メリット提示型**
   - 「日勤のみ・残業少なめの正社員求人のご案内」
   - 「資格取得支援あり｜キャリアアップを目指す方へ」

3. **疑問形型**
   - 「〇〇エリアでの転職をお考えですか？」

## 本文の構成

### 1. 導入（2〜3文）

**含めるべき要素:**
- 挨拶と自己紹介（会社名・担当者名）
- スカウトを送った理由（プロフィールのどこに注目したか）

**例:**
```
はじめまして。〇〇会社の採用担当△△と申します。
〇〇様のプロフィールを拝見し、特養での3年間のご経験と
介護福祉士の資格に魅力を感じ、ご連絡いたしました。
```

### 2. 魅力提示（3〜5文）

**含めるべき要素:**
- 会社・施設の特徴（1〜2点に絞る）
- 求職者にとってのメリット
- 具体的な数字や事例

**例:**
```
当施設は開設5年目の特別養護老人ホームで、
入居者様30名に対しスタッフ20名体制で運営しております。
残業は月平均5時間以下、有給消化率は80%を超えており、
プライベートとの両立を大切にする方に選ばれています。
```

### 3. CTA（1〜2文）

**含めるべき要素:**
- 次のステップを明確に
- ハードルを下げる言葉
- 返信期限は設けない（プレッシャーを与えない）

**例:**
```
まずはオンラインでの施設説明も可能ですので、
ご興味がございましたらお気軽にご返信ください。
```

## 適切な文字数

- **全体**: 300〜500文字程度
- **件名**: 30文字以内
- **導入**: 50〜100文字
- **魅力提示**: 150〜250文字
- **CTA**: 50〜100文字

## 読みやすさのポイント

- 1文は60文字以内に
- 適度な改行を入れる
- 箇条書きは最小限に（メールらしさを保つ）
