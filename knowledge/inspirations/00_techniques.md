# スカウト文テクニック集（業界横断）

inspirationsフォルダの事例から抽出した汎用パターン。

---

## 1. 開封させる（件名・冒頭）

| テクニック | 効果 | 例 |
|-----------|------|-----|
| 件名にブランド企業名を並べる | 知名度で開封率UP | 「日立/東京エレクトロン/双日」 |
| 件名を候補者の願望で書く | 自分ごと化 | 「ワークライフバランスを実現しながら〜」 |
| 冒頭3行で「なぜあなたか」を出す | 読む価値の即時判断 | スカウト理由を箇条書きで先出し |
| 「単刀直入に申し上げます」型 | テンプレ感の排除、注意喚起 | 前置きを飛ばして本題 |

## 2. 「あなただから送った」の明示

パーソナライズ文で暗に伝わっていても、**明示的に「あなた限定」と言い切る1文**があると特別感が段違い。

| パターン | 例 |
|---------|-----|
| 条件限定型 | 「〇〇の資格をお持ちの方に限定してご案内しております」 |
| 経歴注目型 | 「どうしてもご経歴が目に留まり、ご連絡させていただきました」 |
| 選別済み型 | 「条件合致する方だけにお送りしています」 |
| 直感型 | 「ぜひ弊社でご活躍いただきたく、お声がけさせていただきました」 |

**→ テンプレート冒頭 or パーソナライズ文の直前に1行差し込むだけで導入できる。型はめパターンにも組み込みやすい。**

## 3. パーソナライズの深度

| レベル | やり方 | 効果 |
|--------|--------|------|
| Lv.0 | 「プロフィールを拝見し」のみ | ほぼ効果なし |
| Lv.1 | スキル名を列挙 | 「一応読んだ」感 |
| Lv.2 | スキル×経験→ポジション適性を論理展開 | 「自分のために書いた」感 |
| Lv.3 | 過去経験→将来像まで接続 | 「キャリアを考えてくれている」感 |

**→ 我々のスカウトはLv.2〜3を型はめ+AIで実現している。この強みは維持すべき。**

## 4. 返信ハードルを下げる

| テクニック | 使い方 |
|-----------|--------|
| 「面談希望」の一言でOK | 返信文を考えるコストをゼロに |
| 「今すぐの転職でなくてもOK」 | 転職意欲が低い層も取り込む |
| 「情報収集だけでも歓迎」 | コミットメントの重さを排除 |
| 「書類選考免除」 | 選考プロセスのフリクション除去 |
| 「志望動機不要・服装自由」 | 面談そのものの心理コスト削減 |
| 面談時間を明示（20分〜30分） | 時間コミットの見積もり可能に |

**→ 医療介護スカウトでも「まずはお気軽に」系は使えるが、具体化が足りていない可能性あり。「見学だけでもOK」「LINEで質問だけでも」等に翻訳できる。**

## 5. 証拠で語る

| 証拠の種類 | 効果 | 例 |
|-----------|------|-----|
| 年収Before/After事例 | 自分に重ねてイメージできる | 年齢×スキル×入社後推移 |
| 第三者評価の数値 | 自社の主張より信頼される | OpenWork上位1%、各指標の業界平均比較 |
| 社員インタビュー記事 | リアルな声で不安を軽減 | note記事リンク複数 |
| YouTube対談 | テキストでは伝わらない雰囲気 | 社長×有名人の対談動画 |
| 定着率・残業時間 | 働きやすさの定量的裏付け | 定着率91%、残業6.8h |

**→ 医療介護では「職員の声」「見学動画」「定着率」が対応する。profile.mdに数値があれば積極的に使う。**

## 6. 構造・フォーマット

| テクニック | 効果 |
|-----------|------|
| 悩み代弁→解決提示の構成 | 「分かってくれている」感 |
| 求人カタログ形式（複数提示） | 1つ外してもどれかに刺さる |
| 統一フォーマットで求人比較 | 読み手の認知負荷を下げる |
| セクションを罫線・見出しで分離 | スクロール中でも構造が分かる |
| 「他社スカウトへの不安」を代弁 | 競合と差別化しつつ共感を得る |

## 7. 再送（フォローアップ）

| テクニック | 効果 |
|-----------|------|
| 再送であることを冒頭で明示 | 誠実さ、しつこさの緩和 |
| スカウト理由を再掲＋新情報追加 | 前回と同じ文面ではない価値 |
| 前回より短く | 2回目は要点だけ |

**→ 我々の再送テンプレートにも「前回からの新情報」（新求人、条件変更等）を入れる余地あり。**

## 現行スカウトへの示唆

| 我々が既にやれていること | まだ取り込めそうなこと |
|------------------------|----------------------|
| パーソナライズの深度（Lv.2-3） | 「なぜあなたか」を冒頭に持ってくる構成 |
| 職場の具体情報（profile.md） | 定量的な証拠の強化（定着率、残業時間等） |
| テンプレートの職種別最適化 | 返信ハードルの更なる低減（「見学だけ」「LINEで質問」等） |
| 型はめによる安定品質 | 再送時の差分情報設計 |
