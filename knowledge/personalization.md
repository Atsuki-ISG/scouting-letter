# パーソナライズのコツ

## プロフィールから読み取るべき情報

### 基本情報

- 氏名（スカウト文で使用）
- 居住エリア（通勤圏内かどうか）
- 希望職種・雇用形態
- 希望勤務地

### 経歴・スキル

- 保有資格（介護福祉士、ヘルパー2級、看護師など）
- 経験年数
- 経験した施設形態（特養、デイ、訪問、病院など）
- 具体的なスキル・得意分野

### 希望条件

- 給与・待遇の希望
- 勤務時間・シフトの希望
- 転職理由（記載がある場合）
- 重視するポイント

### その他の手がかり

- 自己PR文の内容
- 資格取得の時期や経緯
- ブランクの有無と理由

## パーソナライズの具体的手法

### レベル1: 名前・資格の言及

**最低限のパーソナライズ**
```
〇〇様の介護福祉士の資格を拝見しました
```

### レベル2: 経験への具体的言及

**経歴を読み込んでいることを示す**
```
特別養護老人ホームで3年間ご勤務された経験、
特に認知症ケアに携わってこられた点に注目しました
```

### レベル3: 希望条件とのマッチング

**相手のニーズに応える姿勢**
```
日勤帯でのお仕事をご希望とのこと、当施設では
デイサービスでのポジションがございます
```

### レベル4: キャリアビジョンへの共感

**将来を一緒に考える姿勢**
```
ケアマネージャーを目指されているとのこと、
当社では資格取得支援に加え、取得後の
キャリアパスも明確に用意しております
```

## 刺さるポイントの見つけ方

### 転職理由別アプローチ

| 転職理由 | 刺さるポイント |
|---------|--------------|
| 給与アップ | 具体的な年収例、昇給実績 |
| ワークライフバランス | 残業時間、有給消化率、シフト融通 |
| キャリアアップ | 研修制度、資格支援、昇進実績 |
| 人間関係 | チームの雰囲気、離職率、サポート体制 |
| 通勤 | アクセス、車通勤可、駐車場完備 |

### プロフィールの「行間」を読む

- **資格取得時期が最近** → 学ぶ意欲が高い、実践経験を積みたい
- **複数施設を経験** → 幅広い経験を活かしたい、または合う職場を探している
- **ブランクあり** → 復帰への不安、サポート体制を重視
- **詳細な自己PR** → 熱意が高い、しっかり読んでほしい

## 注意点

### やりすぎ注意

- 情報を羅列しすぎない（1〜2点に絞る）
- 決めつけない（「〜されたいのですね」より「〜をお考えでしたら」）
- プライバシーへの配慮（年齢や家族構成への直接言及は避ける）

### 情報がない場合

プロフィール情報が少ない場合は:
- **会社の強み（教育体制、成長環境など）を前面に出す**
- 地理的要素（希望勤務地の一致）のみに頼らない
- 質問形式で興味を引く
