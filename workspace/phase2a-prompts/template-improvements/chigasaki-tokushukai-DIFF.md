# 茅ヶ崎徳洲会病院 templates.md 改善 DIFF

## (a) 何を変えたか／なぜ

### 1. ヘッダーの差別化強化

**Before**（初回のみ・件名として分離 & ヘッダー未使用）:
件名: `【茅ヶ崎駅徒歩5分】時短勤務・残業時間20時間以下の働きやすい病棟`
本文先頭: なし（「はじめまして！」からスタート）

→ 「時短勤務・残業20時間以下」は業界で普通の訴求で差別化ゼロ。独自強みが preview に表示されない。

**After**（3テンプレ共通のヘッダー本文）:
```
■茅ヶ崎駅徒歩5分｜新卒離職率10年連続0%／中途3ヶ月マンツーマン／24時間託児所10歳まで／医療費減免 家族まで
```

理由:
- 競合調査で特定した独自強み4本を preview 80字に凝縮
- `■` は病院フォーマルトーン（header-personalization-matrix §4）
- 「時短勤務」「残業20h以下」は相場並 → 独自性のある新卒離職率・託児所10歳・医療費減免に差替え

### 2. お気に入り件名の刷新

Before: 件名ブロックは本文内に組み込まれた1テンプレのみの簡素な構造（実質お気に入り専用件名なし）

After: 専用件名を追加。
```
【面談確約】「気になる」を押してくださり、ありがとうございます！新卒離職率10年連続0%／中途3ヶ月マンツーマン／24時間託児所10歳まで／医療費減免 家族まで対象
```

理由: LCC と ARK のお気に入り件名と同じパターンに揃え、JM側で件名別管理ができる構造に。

### 3. Distinctive value セクションの追加

Before: 「スタッフが安心して働けるよう、事前の『院内見学・説明会』を積極的に受け入れており、ご希望の方には入職後のミスマッチを防ぐための簡易的な業務体験の機会もご用意しています」のみ。病院の独自性が伝わらない。

After: 下記の1パラグラフを挿入:
```
当院は、急性期と地域包括ケアの2つの病棟で二次救急を担う144床の地域病院です（2026年4月から172床に増床予定）。教育面では新卒看護師の離職率が10年連続0%、中途入職の方には専任担当者が3ヶ月間マンツーマンでサポートに入ります。「経験者だから放置する」ことは絶対にしない、というのが院全体の方針です。2015年、地域住民6,372名の署名運動で守られた病院、というのが私たちの原点でもあります。
```

理由: 競合調査の独自強み5本（急性期+地域包括2本柱／172床増床／新卒0%・中途マンツーマン／住民6,372名署名ストーリー）を物語として凝縮。徳洲会グループのネガ評判（OpenWork 2.9）を先取りで打ち消す文脈になる。

### 4. 募集要項概要の構造化（LCC方式）

Before: `📍勤務地 / 📍仕事内容 / 📍給与 / 📍特徴 / 📍勤務時間 / 📍ご希望の働き方をご相談可能` の6ブロック。`📍特徴` にbullet spam 4行、勤務時間ブロックの中に「救急搬送1日6-7件」が埋没。

After: 仕事内容／給与／勤務時間／休日／福利厚生／所在地の6セクションに整理。
- 勤務時間にあった救急搬送情報は「仕事内容」に集約
- 「特徴」の4項目（賞与3ヶ月／医療費減免／託児所／時短）を福利厚生セクションに分離し、**bold + 説明文**で詳細化
- 給与は月給（基本給＋諸手当）・想定年収・賞与・手当を独立行で列挙（profile.md 数字に忠実）
- 休日セクションを独立させ、年休110日＋リフレッシュ休暇4日のカバーセットを明示
- 所在地とアクセスを末尾に独立

### 5. 独自福利厚生の具体化

Before:
- 「家族の医療費も病院が負担！『医療費減免制度』」の1行
- 「24時間託児所。0歳から10歳まで利用可能な託児所を完備！」の1行

After:
- `**医療費減免制度**：グループ病院での月々の保険診療費 自己負担3,000円超過分を会社負担（職員本人＋一等親以内の家族まで対象）` — 具体金額基準と対象範囲を明示
- `**24時間託児所**：0歳〜10歳まで、スポット利用1日1,000円、金曜は夜勤時もお預かり可` — 料金とスポット/夜勤利用の具体を追記（profile.md 記載）
- `**時短勤務**：3歳前日まで、通常勤務から-2時間まで柔軟（前倒し・後ろ倒し両対応）` — profile.md の「前倒し・後ろ倒し」柔軟性を追加

### 6. 再送テンプレートの新設

Before: 再送テンプレートが存在しない（初回のみ）

After: LCC 方式の再送テンプレートを追加。
- 冒頭「度々のご連絡、大変失礼いたします。〜諦めきれず再度ご連絡させていただきました。」
- 2文目「改めてご経歴と茅ヶ崎を希望勤務地に〜」
- 本文・募集要項は初回と同一

理由: 初回反応なしの候補者への再アプローチは他社と同等の標準フローとして必要。

### 7. 成長機会の訴求追加

Before: 172床増床への言及なし

After: 「2026年4月172床増床に伴い、新病棟の立ち上げメンバーも募集」を仕事内容に追記。Distinctive value パラグラフにも盛り込み。

### 8. 育休復帰率80〜100%の明示

Before: 本文中に記載なし

After: 休日セクションに「育休復帰率80〜100%」を追加（業界平均60-80%より高い数字、競合調査§5）

### 9. お気に入り本文の structural upgrade

Before: お気に入り本文は初回と同じ「📍特徴 bullet 4行」のまま。

After: お気に入り本文もLCC方式の7セクション構造に刷新（初回と共通）。
