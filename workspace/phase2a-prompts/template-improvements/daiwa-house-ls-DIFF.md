# 大和ハウスLS（ネオ・サミット湯河原）templates.md 改善 DIFF サマリー

## (a) 何が変わったか・なぜ

### 構造の刷新（LCC gold standard 準拠）

- **現行**: 入居相談員 正職員 初回／再送／お気に入り の3本（現行で既に用意されている）。ただしタイトル・本文とも「入居募集の営業活動や見学会のご案内、ご契約対応」の繰り返しで、独自強みの訴求が弱い。
- **改善**: 同じ3本構成を維持しつつ、LCC と同じ `仕事内容／給与／勤務時間／休日／福利厚生／所在地` のクリーンなセクション構造に差し替え。

### ヘッダーを preview 最適化

- **現行タイトル**:
  - 初回: `【面談確約】入居募集の営業活動や見学会のご案内、ご契約対応をいただける入居相談員募集！静岡県で生活相談員の経験を活かしませんか？`
  - 問題: 「面談確約」以外の訴求がタスク内容の繰り返し。JMのpreviewで最も目立つ「冒頭80字」が凡庸。静岡県・生活相談員という切り口も弱い。
- **改善後ヘッダー**:
  - 初回／再送／お気に入り 共通: `【熱海・湯河原Uターン歓迎｜相模湾一望のリゾート立地】転居補助70万／営業奨励金年50万／入社初日から有給／10-20年伴走する営業`
  - **競合研究§7 で発見した「介護業界で上位水準」の数字と「伴走型営業」という差別化コンセプト**を前面に。
  - Uターン・Iターン層向けに「熱海・湯河原」の具体地名と「リゾート立地」を冒頭に。

### 求人要項部分のスマート化

- **現行の問題点**:
  - 「特徴：☆」が7行並び、給与が月給範囲・想定年収の2行だけ
  - 勤務地が「〒413-0001 静岡県熱海市泉17-2 ネオ・サミット湯河原」とやや冗長
  - 「奨励金500,000円」「転居補助40-70万」「入社初日有給」「10-20年伴走モデル」「大和ハウスグループ割引」等の独自強みが「特徴：☆」の中で横並びに埋没
- **改善後**:
  - 仕事内容／給与／勤務時間／休日／福利厚生／所在地 の6セクションで整理
  - 給与セクションで「職能給＋役割給＋各種手当」の構造を明示（月収範囲の根拠を可視化）
  - 奨励金「年10件達成で50万円（月1件ペース）」を給与セクションに昇格
  - 転居費用補助（単身40万・家族同伴70万）は福利厚生セクションに明示
  - 「10-20年伴走する営業」を本文中段に独立した訴求パラグラフとして配置

### 独自強みの織り込み

- 競合研究§7 の「隠れた強み」5つを具体的に反映:
  - ①10-20年伴走する営業モデル → 本文ヘッダー＋中段パラグラフ
  - ②転居費用補助（単身40万・家族70万） → ヘッダー＋福利厚生
  - ③入社初日から有給 → ヘッダー＋休日
  - ④大和ハウスグループ割引 → 福利厚生
  - ⑤営業奨励金 年10件で50万（月1件ペース・到達可能性訴求） → ヘッダー＋給与
- 物件の商品力（相模湾一望／温泉大浴場／麻雀・カラオケ・芝生広場）を中段パラグラフで「営業時の強みになる」文脈で訴求

### パーソナライズポイントの拡充

- 介護業界経験者 5パターン＋他業界営業経験者 3パターン＋未経験・転身 2パターンに整理
- 不動産営業・保険営業・リゾート営業からの転身層への接点表現を追加（競合研究§10・§7①）
- Uターン・Iターン志望者向けに転居補助で呼び込む接点を明示

## (c) [要確認] 項目 (PM 宛て)

1. **営業奨励金の達成者比率** — 「年10件達成で50万円」の到達可能性をもっと訴求したいが、過去3年の達成者実績が profile.md・求人票に記載なし（競合研究§11-7）。
2. **奨励金10件超過分の追加インセンティブ** — 設計があるか否か未確認。あれば上振れ訴求できる。
3. **転居費用補助の利用実績・最低勤続要件** — 「転居補助70万」の説得力を高めるため、過去実績と返還条件の確認が必要（競合研究§11-4,5）。
4. **現場裁量の実態** — 大和ハウスライフサポート全体の口コミ（エンゲージ3.1点）で「本社と現場の温度差」が指摘されているため、ネオ・サミット湯河原固有の運営独立性をどこまで訴求できるか要確認（競合研究§11-3）。本テンプレでは踏み込まず、ヘッダー・本文中で施設固有の強みに絞る形にした。
5. **住宅棟→ケア棟の移行率** — 「10-20年伴走する営業」の根拠として、移行率・平均入居年数データが求人票・profile.md に未記載（競合研究§11）。
6. **大和ハウスグループ割引の具体内訳** — 対象ブランド名・割引率・上限が profile.md に未記載。「ホテル・スポーツクラブ・ゴルフ場等」の例示に留めた。
7. **入社初日有給の日数** — profile.md「入社初日から有給休暇付与」のみ。日数が未確認（競合研究§11）。
