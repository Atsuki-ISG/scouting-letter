# ARK訪問看護 templates.md 改善 DIFF

## (a) 何を変えたか／なぜ

### 1. ヘッダーの差別化強化

**Before（全テンプレ共通）**:
```
【札幌市西区｜認定看護師2名の指導体制】併設クリニック医療費還付／子育てママ多数／ブランクOK
```
→ パート・正社員で同一で、雇用形態ごとの強みが伝わらない。「認定看護師2名」と抽象的。

**After**:
- パート共通: `【札幌市西区｜認知症ケア・がん認定看護師2名が現場指導】併設クリニック医療費全額還付／残業ほぼなし／子育て・ブランクOK`
- 正社員共通: `【札幌市西区｜認知症ケア・がん認定看護師2名が現場指導】併設クリニック医療費全額還付／退職金・研修費年5万円補助／残業ほぼなし`

理由:
- 「認定看護師2名」→「**認知症ケア・がん**認定看護師2名が**現場指導**」で指導領域と関与度を明示
- 「医療費還付」→「医療費**全額還付**」で独自性を強調
- 雇用形態ごとに後半セレクションを分岐（パート＝柔軟性／正社員＝待遇・教育投資）

### 2. Distinctive value セクションの追加

Before: 「地域に根差した当訪問看護ステーションが目指す『地域の皆様に寄り添う訪問看護』において、重要な役割を担っていただけると感じております」のみの抽象文 → 削除

After: 下記の1パラグラフを挿入:
```
当ステーションは、利用者様の約8割が認知症をベースとする慢性疾患の方で、年20件前後の在宅看取りにも向き合っています。だからこそ、認知症ケア認定看護師・がん看護認定看護師の2名が日常的に現場でのOJT指導に入り、医療法人母体の併設クリニックとも一体で連携できる体制を整えています。新設ステーション（2023年開業）ながら、医療法人の基盤の上で、スタッフの声を反映しながら制度を育てている最中です。
```

理由: 競合調査で特定した独自強み3本（認知症8割×看取り年20件／認定看護師2名の現場OJT／医療法人母体×新設の柔軟性）を一度に物語として語る。

### 3. 募集要項概要の構造化（LCC方式）

Before: 「特徴:」ブロックに「☆ 成長環境◎」「☆ 賞与・昇給あり！」など☆×9行のbullet spam。情報が同じ箱に混在し読みづらい。

After: 仕事内容／給与／勤務時間／休日／福利厚生／教育体制／所在地の7セクションに分離。
- 独自ベネフィット「併設クリニック医療費全額還付」は福利厚生セクションの筆頭に
- 「認定看護師が現場指導」は教育体制セクションに
- 給与には「札幌市西区で上位水準／相場上限」の相対情報を追加

### 4. 3〜5月非採用期の告知位置変更

Before: 本文3段落目（増員時期の説明が前半に来て、興味喚起を妨げる）

After: CTA直前の案内文として末尾に移動（興味を持った後の実務注意としての提示）

### 5. 訪問範囲の追記

Before: 勤務地（住所）のみ。

After: 「訪問範囲：札幌市西区・手稲区・北区・中央区（東区・豊平区）」を追加（profile.md §訪問エリア）。

### 6. 既存の冗長ブロックの削除

Before: 「現在、アーク訪問看護ステーションでは、札幌市西区でご勤務いただける看護師を募集しております。」という告知文と「当看護ステーションは、ワークライフバランスを重視しており〜無理なく長期的に活躍できる環境を提供します。」という抽象文が3段落連続。

After: Distinctive value パラグラフ1つに集約。ワークライフバランスは「給与・勤務時間・休日・福利厚生」の具体データで語らせる。

### 7. お気に入りテンプレの件名更新

Before: 「賞与・昇給あり！充実した研修制度・費用補助◎クリニック併設で営業ノルマなし！」等のbullet集。

After: 雇用形態別に独自強みを凝縮。
- パート: 「認知症ケア・がん認定看護師2名が現場指導／併設クリニック医療費還付／Wワーク・ブランクOK」
- 正社員: 「認知症ケア・がん認定看護師2名が現場指導／併設クリニック医療費還付／研修費年5万円補助」

### 8. リハ・医療事務セクションは追加しない

ARK は看護師のみ対象（profile.md＋既存テンプレ確認）。看護師パート／正社員／お気に入り（パート）／お気に入り（正社員）の4種のみを改善。

## (c) [要確認] 項目（PM→クライアント黒木氏へヒアリング）

### テンプレ本文に明示で [要確認] マーカーを残した箇所

- **年間休日日数**: 競合の多くが年休120日を明示。ARK は profile.md・既存テンプレ共に未記載。4テンプレ全てに `[要確認：年間休日日数]` を残した
- **オンコール回数・手当**: 正社員版のみ `[要確認：オンコール回数・手当]` を残した（看護roo! 等で業界相場 月2-3回＋待機1-3K/回または月固定1.5-2万円が標準）

### 担当者名の不一致

- プロンプト指定: 古泉
- profile.md: 黒木 慎一（代表）
- 既存 templates.md 末尾も黒木 慎一

→ profile.md を信じて **黒木 慎一** を維持。古泉氏に移管するならサーバ push 前に担当者名を確認・差替え必要。

### スカウト文に入れる前にクライアント確認が望ましい数字

- 医療費還付の年間上限・対象診療科・家族含まれるか（本文では「自己負担分全額還付」で表現）
- 認定看護師2名の現場OJT関与頻度（週何回、同行か、カンファか）
- 育休復職率実績（現状「取得実績あり」のみ）
- 新人同行訪問期間（現状「入社初日から新任研修制度あり」のみ）

競合調査 §11 に25項目の詳細な逆質問リストあり。上位4項目は本テンプレートで直接使われる数字なので優先度高い。
