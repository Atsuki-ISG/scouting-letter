# 野村病院 templates.md 改善 DIFF

## (a) 主な変更点と理由

### 1. ヘッダーの新設（そもそもヘッダーがなかった）
**Before**: テンプレ冒頭がいきなり「はじめまして。突然のご連絡大変失礼いたします、医療法人天秋会 野村病院の採用担当と申します」から始まる。**preview に差別化情報が一切載らない設計**（JMプレビュー80字を挨拶で潰している）。

**After（看護師）**: `■稼働70床の慢性期精神科｜週2日〜・1日4時間〜の正社員／定年75歳まで／ビデオオフOKオンライン説明会`（約55字、病院フォーマル■、独自強み3-4点に凝縮）

**After（管理栄養士）**: `■栄養課室勤務・調理/厨房ほぼなし｜夜勤なし・残業ほぼなし／賞与年2回計3ヶ月／定年75歳まで`（約48字）

**理由**: 競合調査で「経営立て直し中の開示が不安を生む」「18.6〜22.1万が最初に目立って見劣り」と指摘されていた。**給与競争を避け、独自強み（柔軟勤務・長期キャリア・心理ハードル最小の採用手法）を前面に**。看護師と管理栄養士でフックを明確に分け、**管理栄養士テンプレは看護師要素（夜勤・LGBTQ+・介助業務）を一切混入させない**（既存ルールを遵守）。

### 2. 給与訴求順を刷新
**Before（看護師）**: 給与 `月給 186,000円 〜 370,000円` を冒頭に書いた上で、カッコ内に「管理職候補30〜37万、一般常勤18.6〜22.1万」と並列説明。**最初の数字186,000円が最も弱い数字**で求職者の離脱要因になっていた（競合調査§9指摘）。

**After**: 
```
・月給：186,000円〜370,000円（経験・働き方により決定）
・管理職候補：月給300,000〜370,000円
・一般常勤看護師：月給186,000〜221,000円
・夜勤手当6,500円/回、夜勤専従（月10〜11回）可、住宅手当・家族手当あり
```
**管理職候補を先に列挙**し、夜勤専従という稼げる選択肢も明示。総支給としての選択肢の広さが伝わる構造に。

### 3. 「経営立て直し中」をDistinctive valueで言い換え
**Before**: 本文に「東京医療法人バックアップで経営立て直し中」の明示はなし（profile.md内部情報のみ）。だが候補者が別ルートで知ると不安要因になる。
**After**: 代わりに **「稼働70床の慢性期精神科、一人の看護師が関わる患者数を抑えた」「介助8割＋処置2割」** という物理的・業務的な落ち着きを前面に出し、**「急性期疲弊層」の受け皿としてポジショニング**。経営再建のネガ情報には触れず、変革期のポジ要素（柔軟勤務・管理職ポスト）のみ構造的に訴求。

### 4. 求人要項を LCC 構造に整理
**Before**: 「特徴：」セクションに ☆付き bullet 6行、その後に「勤務時間」「勤務地」のみというアンバランスな構成。「福利厚生」「休日」「応募要件」が分散。
**After**: `仕事内容 / 給与 / 勤務時間 / 休日 / 福利厚生 / 所在地・アクセス` の6セクション統一。同じ情報（例: 時短勤務相談可、副業OK、車通勤可）を複数箇所に書かない。

### 5. 「週2日〜・1日4時間〜の正社員」を強く押し出す
**Before**: profile.md に記載はあるが、テンプレ本文では一切触れられていなかった（「日勤のみ・2交代・夜勤のみ」のbullet1行のみ）。
**After**: ヘッダー・Distinctive value・勤務時間セクションの3箇所で訴求。**「正社員＝フルタイム」の固定観念を外しWワーク層・育児両立層・50-60代復職層を呼び込める** 強力なフック（競合調査§7強み③）。

### 6. 「オンライン説明会 ビデオオフOK・書類不要」をCTAで活用
**Before**: 特徴bullet に「事前の院内見学や説明会が可能」と抽象的に1行。
**After**: 各テンプレ末尾CTAで「**ビデオオフOK・応募書類不要・平日毎日開催**」を明記。**返信ハードル最小化を具体訴求**（競合調査§7強み⑤）。

### 7. 「定年75歳まで」を明示
**Before**: profile.md に「定年65歳、再雇用75歳まで」と記載があったが、テンプレ本文では一切訴求なし。
**After**: ヘッダー＋福利厚生セクションで明示。50-60代復職層・長期キャリア志向層向けの強力フック（競合調査§7強み④）。

### 8. 「介助8割＋処置2割（軟膏塗り・服薬・採血がメイン）」で急性期疲弊層にアピール
**Before**: profile.md に明記されていたが、テンプレ本文では「精神科病棟における看護業務全般」と抽象化。
**After**: Distinctive value と仕事内容セクションの両方で明示。**侵襲的処置の少なさ = 急性期疲弊層にとっての安心材料** を事実ベースで訴求。

### 9. LGBTQ+フレンドリー訴求の適切な配置
**Before**: profile.mdに「LGBTQ+フレンドリー」明示があるが、テンプレ本文では一切言及なし。
**After**: 看護師テンプレの福利厚生セクションに「副業OK／LGBTQ+フレンドリー」として明示。ただし**管理栄養士テンプレには含めない**（看護師特化情報の混入禁止ルール遵守。ただし採用方針としては全職種適用が自然なので、PM確認後に判断推奨 → [要確認]）。

### 10. 管理栄養士テンプレの雇用形態を「契約職員」→「正社員」に修正
**Before**: 「雇用形態：契約職員」と記載。
**After**: profile.md §管理栄養士 の記述（「職種：管理栄養士/栄養士（正社員）」「定年65歳（再雇用75歳）」「試用期間3ヶ月」「退職金制度あり（勤続2年以上）」「賞与年2回 計3ヶ月分」）に基づき **正社員として記述**。旧テンプレは「契約職員・産休代替」の扱いだった可能性があるが、profile.mdの現状記述は正社員扱い。**要PM確認**（[要確認]項目で明記）。

### 11. 「管理栄養士＝栄養課室中心・調理/厨房ほぼなし」を前面に
**Before**: 特徴bullet に「管理栄養士資格があれば業務経験は問いません」等の抽象情報。
**After**: Distinctive value + 仕事内容セクションで「**勤務場所は栄養課室、調理・厨房業務・介護業務はほぼなし**」を前面に出す。これは profile.md §管理栄養士仕事内容 の明示ポイントで、**専門職としての業務に集中できる環境** という強い訴求材料。

### 12. お気に入りテンプレの件名＋本文分離を完全整備
**Before**: 件名のみ記述、本文は「初回テンプレートと同じ」の参照形式。
**After**: 2職種分の件名＋本文を両方ともコードブロックで完全記述。LCC・an と同じ構成で、感謝文＋「面談確約」を明示するお気に入り専用導入を使用。

### 13. 「拝見禁止」ルールを生成ルール冒頭で明文化
既存の「テンプレ冒頭に『ご経歴を拝見し』があるため、パーソナライズ文は『拝見し』以外の書き出しにする」ルールを引き続き維持。

## (c) [要確認] 項目（PMヒアリング推奨）

1. **[要確認・最優先] スカウト担当者名** — profile.md §基本情報 では「（未定）」。今回テンプレは現状維持で「採用担当」としたが、**看護師長や事務長など実名に置換することで信頼感向上**（競合調査§11-17, §9）。誰を出すか要確定。
2. **[要確認・最優先] 管理栄養士の雇用形態** — 旧テンプレ「雇用形態：契約職員」と profile.md「正社員」で記載が食い違う。現在の募集は **正社員** か **契約職員（産休代替）** かを確定。新テンプレは正社員前提で作成。契約職員なら仕事内容・給与・勤務時間は同じでも「契約期間」「契約更新条件」の追記が必要。
3. **[要確認] 稼働70床の理由・期間** — 「医師不足」「意図的な負荷調整」「経営再建中の判断」のいずれか。今後稼働率を上げる計画があるか（上がると負荷増）。「じっくり関われる」訴求の持続性に直結（競合調査§11-6〜8）。
4. **[要確認] 「週2日正社員」の実績** — 現在この制度で働いているスタッフが実在するか。短時間正社員とフルタイム正社員で賞与・昇給に差があるか（競合調査§11-12〜13）。
5. **[要確認] LGBTQ+フレンドリーの具体施策** — 「呼称の自由・休暇制度・差別禁止方針」等の具体内容。明示だけでは真剣度が伝わらない（競合調査§11-14）。管理栄養士テンプレへの明示有無も要判断。
6. **[要確認] 一般常勤看護師 月給18.6〜22.1万円の内訳** — 手当込みの総支給か基本給のみか。夜勤専従での月収実例。総支給の見せ方が変われば訴求軸が変わる（競合調査§11-9〜10）。
7. **[要確認] 年間休日107日のカバー** — 相場115〜120日より少ない。家庭都合休・夜勤明けの実質休日を含めた「実質休日数」を打ち出せるか（競合調査§9改善候補）。
8. **[要確認] 東京医療法人・コンサル会社名の公表可否** — 今回テンプレでは経営再建を意図的に触れなかったが、将来の信頼材料として必要（競合調査§11-1〜3）。
9. **[要確認] 75歳までの再雇用実例** — 実際に75歳まで働いている看護師の有無。実例があれば強力な訴求材料（競合調査§11-25）。
