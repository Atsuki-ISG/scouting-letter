# an訪問看護ステーション templates.md 改善 DIFF

## (a) 主な変更点と理由

### 1. ヘッダーを preview 最適化
**Before**: `🌸看護師募集🌸年収550万円〜！年間休日124日以上！完全週休2日制！9〜17時の7時間勤務！残業ほぼなし！web説明会実施中！`（約60字、職種・給与がJMプレビューと重複、「！」連打）

**After**: `🌸【弘明寺駅徒歩3分｜9-17時の7時間勤務】残業月5時間／年休124日／看護+リハ+相談支援の3事業連携`（約52字、具体アクセス＋独自強みに凝縮）

**理由**: 職種（看護師）・給与レンジ（550万円〜）・完全週休2日はJMプレビューで既に表示される。重複を排し「9-17時の7時間勤務」「残業月5時間」「3事業連携」といった **競合10社で未確認の独自強み** をヘッダーに集約。「！」連打の求人チラシ風味を排除。

### 2. 求人要項の「bullet spam」を整理
**Before**: 「特徴：」セクションに5〜6個の ◎ 付き重複bullet、「勤務時間と定休日：」「教育体制・研修」「勤務地：」など項目バラバラに展開。
**After**: LCC構造の `仕事内容 / 給与 / 勤務時間 / 休日 / 福利厚生 / 所在地` の6セクションに整理。同じ情報（例: 弘明寺駅徒歩3分、年休124日、9-17時）を2回以上書かない。

### 3. Distinctive value セクションを1段落で強化
**Before**: 「地域の皆様に寄り添う訪問看護」という汎用句2-3段落。どの会社にも使える一般論。
**After**: 1段落に凝縮して、**「9-17時の7時間勤務」「3事業一体運営」「1日4〜6件の訪問件数」** という an 固有の構造的強みを事実ベースで説明。「訪看単体の競合から差別化」という競合調査の結論を反映。

### 4. オンコール月固定30,000円を訴求
**Before**: 求人票側では「オンコール手当 月固定30,000円」が月給内訳にあっただけで、テンプレ本文では訴求していなかった。
**After**: 「回数型でなく定額で収入が読みやすい設計」として明記。競合調査§7強み⑤を反映。

### 5. パート時給「訪問インセンティブ込みで時給3,000円相当」の計算イメージを追加
**Before**: 「時給2,000〜2,300円」＋「インセンティブ制度で訪問件数に応じた収入アップが可能」と抽象表現。
**After**: 「40分1,500円/件・60分1,800円/件の訪問インセンティブが付き、訪問件数に応じて時給3,000円相当も目指せる」と具体計算イメージ。競合調査§7強み④を反映。

### 6. 看護師正社員テンプレを新設（不在だった「再送テンプレ」はあったが構造がバラバラだったので統一）
既存: 初回・再送 各職種で個別に書かれていたが、構造（見出し粒度、セクション順）がバラバラ。
改善: 全12テンプレ＋4お気に入り（計16）で **ヘッダー／導入／{ここに生成した文章を挿入}／Distinctive value／募集要項概要／CTA／署名** の統一構造に揃えた。

### 7. お気に入りテンプレの「件名ブロック＋本文ブロック」を完全整備
**Before**: 件名のみ3件、本文は「初回テンプレートと同じ」という参照のみ（Sheets 登録済みの本文は別途記載されていた）。
**After**: 4職種分の件名＋本文を両方ともコードブロックで完全記述。LCCと同じ構成で揃え、感謝文＋「面談確約」を明示するお気に入り専用導入を使用。

### 8. プレースホルダー統一（{ここに生成した文章を挿入}）
an既存の慣習「{ここに生成した文章を挿入}」を全16テンプレで統一維持（LCCの「{ここにパーソナライズ文を挿入}」との違いを保存）。

### 9. パーソナライズ文ガイドから「お気に入り用の感謝文」重複を排除
**Before**: 相談支援専門員向けの接点例が欠落。お気に入り用導入文が2箇所に重複記述。
**After**: 相談支援専門員向け接点を新規追加。お気に入り用導入は各お気に入りテンプレ内に集約。

### 10. 「拝見禁止」ルールを生成ルール冒頭に明記
テンプレ側で「ご経歴を拝見し」と述べるため、パーソナライズ文冒頭での「拝見」使用を禁止する注意を明文化。

## (c) [要確認] 項目（PMヒアリング推奨）

1. **[要確認] 看護師パート時給** — profile.md で未記載、競合調査§9でも「未記載のままパート志望層に訴求できていない」と指摘。今回はパート看護師テンプレを作っていないが、作成するには時給・インセンティブ設定の確認が必要。
2. **[要確認] 「残業月5時間」の算出根拠** — 競合調査§11で「全社員平均か、特定メンバーか／閑散期・繁忙期の変動」が未確認。訴求してよい数字か要確認。
3. **[要確認] 「9-17時の7時間勤務」の制度的根拠** — 就業規則明文化か慣行か（競合調査§11-1）。「制度として明文化」であれば訴求強度を上げられる。
4. **[要確認] 有給消化率ほぼ100%の計測年度** — 最新年度のデータか要確認。
5. **[要確認] オンコール月固定30,000円の内訳** — 待機のみ／出動含むか、月平均当番回数、免除制度の有無（競合調査§11）。
6. **[要確認] 「40-50代活躍中」の具体数** — 今回のテンプレでは候補者のプロフィールにマッチした場合のパーソナライズ側で扱うことを想定し、テンプレ本文では直接言及せず（若年層への誤解防止）。
7. **[要確認] 相談支援専門員の人数体制** — 「相談支援員在籍」とだけ記載されるが1名か複数か。「同じ事業所で連携」という表現の裏付け。
