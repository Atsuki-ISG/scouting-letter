## 変更が適用されたセクション（詳細）

### 1. 冒頭タスク説明（1箇所）

**旧:**
```
候補者のプロフィールを読み取り、会社との接点を見出した一文（約100文字）を作成してください。
```

**新:**
```
候補者のプロフィールを読み取り、会社との接点を見出した文章（情報量に応じて100〜250文字、2〜4文）を作成してください。
```

### 2. トーン＆マナーに 13 を追加

**旧: 1〜12のみ**

**新: 13を追加**
```
13. 【新】会員番号を一切使わない: 「〇〇様」「03066549様」等の番号呼称は禁止。
    経歴の具体性だけで自己認知は成立する
```

### 3. 共通ルール「文字数」の書き換え

**旧:**
```
- 文字数: 初回 約100文字 / 再送 120〜150文字
```

**新:**
```
- 【変更】文字数（AI生成パス）: 情報量に応じて可変。
  自己PR充実 150〜250文字 / 自己PR短めor未入力 100〜180文字。
  過剰な長文も過小な短文も避け、文の完成を優先
```

**理由:** 既存の80-120文字は再送含めても硬直的。あゆみん3通は120-220文字帯で情報量に応じて可変だった。型はめパスは固定文のため影響なし。

### 4. パス2 AI生成に「生成の基本構成」セクションを新設

**旧: 「接点を約100文字で表現する」（4行）のみ**

**新:**
```
#### 生成の基本構成（推奨の段落順）

1. 経歴の具体的言及（役職・年数・分野）— 自己認知成立の土台
2. 自己PRの引用（記載ありの場合のみ）— 下記「引用3ステップ」に従う
3. 業務への橋渡し（候補者の何が × 当社業務の何に）— 説得力の核
4. 簡潔な締め（連絡理由の補強 or 期待）

#### NG構成
- 番号呼び
- 抽象語のみ
- 汎用クロージングのみで締める
- 事実の羅列
```

### 5. 【新規】「引用3ステップ」セクション（AI生成パス内）

Step 1: 引用源の明示はしない（AI臭対策）
Step 2: 導入フレーズを内容に応じて選ぶ（6パターン）
Step 3: 引用の"受け"を内容の重さに合わせる（人生観 / 仕事観 / 具体的工夫 の3温度）

詳細は [system-prompt-new.md](./system-prompt-new.md) の該当セクション参照。

### 6. 【新規】「橋渡し具体化」セクション（AI生成パス内）

「経験を活かせます」→「〇〇だからこそ〇〇できる」に格上げ。
弱い例 vs 強い例の3ペアで方向性を示す。

### 7. 絶対NG表現に3件追加

**旧: 6件**

**新: 9件（3件追加）**
```
- 【新】会員番号呼び（「〇〇様」「03066549様」等の番号呼称）
- 【新】引用源ラベル（「自己PRに綴られていた」「自己PR欄に記載の」等）
- 【新】汎用クロージング単独（「ぜひご活躍いただきたくご連絡いたしました」のみで締める）
```
