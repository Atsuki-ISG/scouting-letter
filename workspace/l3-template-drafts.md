# L3版テンプレート ドラフト集（レビュー用）

**作成日**: 2026-04-24
**対象**: 7社の「初回」テンプレートを L3化（募集要項より前を丸ごとAI生成）
**目的**: Sheets投入前のドラフト精度確認

---

## L3化のルール

全テンプレートで機械的に同じ変換を適用。

| 元L1の部分 | L3での扱い |
|---|---|
| HEADER（🌸/🔶タイトル行の訴求軸） | **固定** — 維持 |
| 挨拶1行（「はじめまして。〜と申します」） | **固定** — 維持 |
| 橋渡し（「この度は、ご経歴を拝見し〜ご連絡いたしました」） | **AI** → `{opening}` に吸収 |
| パーソナライズスロット（`{ここに生成した文章を挿入}` 等） | **AI** → `{bridge}` に吸収 |
| 会社理念／期待の段落 | **AI** → `{facility_intro}` に吸収 |
| 希望条件に合わせた求人訴求の文 | **AI** → `{job_framing}` に吸収 |
| `＜募集要項概要＞` 以下（仕事内容・給与・勤務時間・休日・福利厚生・所在地） | **固定** — 維持 |
| CTA（「ご興味をお持ちいただけましたら〜」） | **AI** → `{closing_cta}` に吸収 |
| 結尾（「ご多忙のところ〜」「何卒〜」） | **AI** → `{closing_cta}` に吸収 |
| 署名 | **固定** — 維持 |

**文字数目安**:
- AI部分: 530〜900字（opening 80-150 + bridge 120-200 + facility_intro 150-250 + job_framing 100-180 + closing_cta 80-120）
- 固定部分: HEADER 30-80 + 挨拶 30-50 + 要項 350-500 + 署名 15-30 = 425-660字
- **合計 約1,000〜1,500字**（現状と同等〜やや長め）
