# スカウト文 候補者別生成サンプル（社内レビュー用）

**作成日**: 2026-04-24
**目的**: 新しい「候補者ごとに募集要項より前をAI生成する」仕組みで作ったスカウト文の品質を、社内のみなさまに確認いただきたい
**フィードバック依頼**: 各サンプルについて「違和感のある表現」「もっとこう書いてほしい」等があればぜひ教えてください

---

## 🎯 この仕組みの狙い（30秒で）

これまでのスカウト文は、候補者ごとに書き換わるのは「パーソナライズ文」と呼ばれる **1段落（100〜180字）だけ** でした。それ以外は全員同じテンプレート固定文です。

新しい仕組みでは、**募集要項より前の部分を丸ごと、候補者ごとにAIが書き分けます**。候補者の経歴・資格・希望条件・こだわり条件に応じて、文体（カジュアル/丁寧/お手紙調など）、訴求する制度（時短、年収、教育体制など）、触れる不安（一人訪問、ブランク復帰、両立 など）を変えます。

狙いは **「あなたのことを考えて送っています」が自然に伝わるスカウト** にすること。押し付けや「汲み取りましたよ」感は避け、**候補者にとって意味のある事実を選んで並べる**ことで、関係性は候補者自身に感じ取ってもらう設計です。

---

## 📦 ブロック構成（簡単に）

募集要項より前の部分を、役割の違う **5つのブロック** に分けて AI が書き分けます。

| ブロック | 役割 | 目安字数 |
|---|---|---|
| **opening（冒頭）** | 候補者の経歴と会社の特色の「接点」を予告する1文。読み手に「私に宛てたスカウト」と瞬時に感じさせる | 80-150字 |
| **bridge（橋渡し）** | その接点が、会社側でどう活きるか・どんな役割を担えるかを展開 | 120-200字 |
| **facility_intro（施設紹介）** | 会社・施設の紹介と、候補者層が気にしそうな領域への具体的な応答（教育体制、OJT、オンコール、WLB 等） | 150-250字 |
| **job_framing（求人フレーミング）** | 求人ポジションを候補者の希望条件に合わせて切り出す | 100-180字 |
| **closing_cta（行動喚起）** | 次のステップへの軽い誘い。プレッシャーなしの低ハードル | 80-120字 |

各ブロックは候補者属性（未経験／ベテラン／ブランク／子育て世代／管理職 等）と文体（カジュアル／コンパクト／ビジネス／お手紙）に応じて書き分けられます。

## 設計の解説（この候補者に対して何を変えたか）

### opening（冒頭接点予告）

```
大学病院の救命救急センターから訪問看護、主任・師長としての病棟運営、そして訪問看護ステーションの立ち上げ責任者までを歩んでこられたキャリアと、感染管理認定看護師の資格が、当院が2026年4月に迎える172床増床の立ち上げ局面と重なる点が多いと存じます。
```

- 候補者の「救命救急 → 訪看 → 主任・師長 → 立ち上げ責任者」という歩みと、「感染管理認定」の資格を **同時に** 取り上げた（どちらも候補者固有の特徴）
- 会社側の接続ポイント: **「2026年4月の172床増床・新病棟立ち上げ」** という、候補者の立ち上げ責任者経験が直接活きる事実を選んだ
- **接点予告まで**に留め、「活きる」「力になる」といった利益表現は次のブロックに委ねた（被りを避けるため）

### bridge（接点の展開）

```
当院は二次救急指定として1日6〜7件の救急搬送を受けながら、急性期病棟と地域包括ケア病棟を固定チームナーシングで運営しています。2026年4月の172床増床に向けて、新病棟の動線設計・院内感染対策・教育体制の整備という局面で、救命救急での臨床知見と感染管理認定の視点をお借りしたいと考えています。DMATメンバーとしてのご経験や訪問診療でのコーディネーター業務は、病棟運営の中でも活きる側面があると考えています。
```

- 冒頭は「当院は〜」で **会社側の文脈から入る**（opening と構造を被せないため）
- 候補者の複数の経験（救命救急・感染管理・DMAT・訪問診療）を**それぞれ別の場面**に当てはめる（一つの経歴の言い換えにしない）
- 「お力をお借りしたい」「活きる側面があると考えています」で**会社側が候補者の経験を必要としている姿勢**を示す（上から目線の「活躍いただけます」や推測での断定「接続可能です」は使わない）

### facility_intro（施設紹介＋不安への応答）

```
中途入職の方には3ヶ月間の専任担当者マンツーマンサポートを設けており、「経験者だから放置」の運用をしない方針で院全体で守ってきました。勤務面は2交代制で月平均残業16〜20時間・夜勤月1〜4回の制度的な上限管理があり、福利厚生面ではグループ病院の医療費減免（本人＋一等親家族まで自己負担3,000円超過分を院負担）と24時間託児所（0歳〜10歳／金曜夜勤時も可）が整備されています。
```

- **ベテラン中途入職者が気にしがちな領域を3つ選んで並べた**:
  1. 新しい職場での孤立不安 → 3ヶ月マンツーマン
  2. 中途＋管理職の勤務負荷 → 残業・夜勤の上限管理
  3. 家族の生活 → 医療費減免＋託児所
- **「あなたのために〜」等の配慮宣言は入れていない**。事実を選んで並べるだけで「わかってくれている」が伝わるよう設計

### job_framing（求人フレーミング）


- PRに書かれていた「感染委員会副委員長」「リンクナース育成」を具体的に引用し、「院内の仕組みづくり」という具体的な貢献先に接続

- **候補者の関心事**（新病棟立ち上げスケジュール）を話題として先取り
- 「見学のみ、カジュアル面談のみでも大歓迎」で低ハードル


- 「〜と重なるように感じました」で接点予告（利益はまだ出さない）

### bridge

```
当ステーションは訪問看護＋訪問リハビリを複数拠点で展開しており、地域包括ケアと訪問看護の両方をご経験されている方には、退院直後のご自宅での暮らしや、ご家族との関係づくりといった場面で、臨床判断と支援調整の両面でお力をお借りしたいと思っています。
```

# フィードバックのお願い

以下の観点で、違和感・改善案があればぜひ教えてください:

## 1. 全体の印象
- 自然な文章として違和感はないか
- テンプレート臭・AI臭はないか
- 「自分に宛てたスカウト」と感じられるか

## 2. 寄り添いの温度感
- 「押し付けがましい／汲み取りましたよ感」はないか
- 逆に「冷たい／事務的すぎる」と感じないか

## 3. 事実の選び方
- その候補者に対して **選ばれている制度・数字** は適切か
- もっと触れるべき訴求軸はないか / 逆に不要な情報はないか

## 4. 構成・読みやすさ
- ブロックの役割分担はわかるか
- 重複・冗長はないか
- 全体の長さは適切か（スマホ開封を考えて）

## 5. 業態別の違和感
- サンプル1（病院・ベテラン）のビジネス調は適切か
- サンプル2（訪看・ブランク育児）の柔らかい調は適切か
