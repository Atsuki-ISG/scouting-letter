# 佐藤病院 v2 ドラフト（committed版の改訂候補）

## 設計原則

茅ヶ崎徳洲会サンプル準拠のテイストを徹底し、以下3ブロックの構成で読み手の脳の負担を減らす。

| ブロック | 該当範囲 |
|---|---|
| **なぜあなたに送っているのか** | 動機提示文 + `{ここに生成した文章を挿入}` |
| **メリット・ベネフィット訴求** | ＼3つのメリット／ ➀➁➂【副題！】箇条書き◎ + 末尾の福利厚生teaser |
| **共感と安心感** | 「中途入職者が持つ〜」共感パラグラフ + ＼歓迎する人／ |

## 改善ポイント（現行committed → v2）

- 動機提示文に「これまでのご経験が」を追加し、候補者要素を一段足す
- ➀➁➂ subtitle を短く punchy に、bullets と重複させない
- 長い引用（「神の時間」スタッフ声）を削除、直接的な価値訴求に置換
- 末尾段落を短縮（「佐藤病院は1919年開設の地域密着型病院で〜」→「職員食堂...嬉しい福利厚生も」）
- 各 bullet 30〜50字で揃え、密度を均一に

**現行committed版・サーバーには反映しない。確認用ドラフト。**
