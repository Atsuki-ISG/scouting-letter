# create-dashboard

Google Sheetsのデータシートからダッシュボードシートを自動生成するスキル。
`_chart_data` 隠しシート + addChart APIで全自動構築。

## トリガー

`/create-dashboard [シート名]` または「ダッシュボード作って」

## 前提

- Google Sheets APIアクセス: `server/sa-key.json` のサービスアカウント
- `GOOGLE_APPLICATION_CREDENTIALS=sa-key.json` で認証（`server/` ディレクトリで実行）
- 対象スプレッドシートにサービスアカウントの編集権限が必要（`scout-api@scout-generation-490709.iam.gserviceaccount.com`）
- 対象スプレッドシートID: `1a3XE212nZgsQP-93phk22VlSSZ5aD1ig72M4A6p2awE`

## アーキテクチャ

```
ソースデータシート（テーブル機能、ドロップダウンバリデーション付き）
  ↓ _chart_data シートのCOUNTIF/SUMPRODUCT数式が直接参照
_chart_data_[会社] 隠しシート（集計数式）
  ↓ addChart APIでグラフ生成
[会社]ダッシュボード シート（KPI + グラフ6枚）
```

### ピボット方式との違い

| 項目 | addChart方式（現行） | ピボット方式（旧） |
|------|---------------------|-------------------|
| グラフ生成 | API自動 | 手動（ピボットチャート） |
| スライサー連動 | なし | あり |
| GAS同期 | 不要（数式直参照） | 必要 |
| データコピー列 | 不要 | T〜AD列に値コピー |
| 保守性 | 高い（全自動再構築可） | 低い（手動ステップあり） |

## ソースデータの前提

### 日付列（応募日）
- **テキスト `M/D` 形式に統一**すること（例: `2/14`, `11/23`）
- Sheetsが自動変換した日付シリアル値が混在していることがある
- ダッシュボード構築前に全シートの日付列をチェック→テキスト変換する
- 書き込みは `valueInputOption='RAW'` で行い再変換を防止

### 職種列
- **お客様行動管理シートの職種名を正とする**
- 各ソースシートのテーブル列はDROPDOWN型に設定（API非対応のため手動）
- ソースデータの既存値も正しい職種名に置換する

| an訪看 | 看護師, 作業療法士, 理学療法士, 相談支援員 |
| アーク訪看 | 看護師, 事務員 |
| いちご訪看 | 看護師(東京), リハ職(東京), 看護師(一宮) |

## 実行手順

### 1. データシート分析

ソースデータの構造を読み取る:
- ヘッダー行の位置と列名
- 選考状況・職種・場所・年齢性別・媒体・雇用形態 等の列位置
- サマリー行（目標/実績/進捗率）の有無と位置
- 日付列の位置と形式（テキスト統一されているか確認）
- データの種類に応じてグラフ6種を決定

### 2. グラフの内容を決定

| 優先度 | グラフ | 条件 |
|--------|--------|------|
| 必須 | 選考状況（縦棒） | 選考ステータス列があれば |
| 必須 | 月別推移（縦棒） | 常に入れる |
| 選択 | 職種別（横棒） | 複数職種がある場合 |
| 選択 | 年齢分布（縦棒） | 年齢・性別列がある場合 |
| 選択 | 男女比（横棒） | 年齢・性別列がある場合 |
| 選択 | 応募経路/媒体別（横棒） | 媒体列がある場合 |
| 選択 | 応募種別（横棒） | 応募種別列がある場合 |

### 3. `_chart_data` 隠しシート作成

各会社に `_chart_data_[会社略称]` シートを作成。COUNTIF/SUMPRODUCT数式でソースを直接参照。

```
=== _chart_data_ark ===
A列: カテゴリラベル  B列: 集計数式

[選考状況セクション]
選考状況  | 人数
書類選考中 | =COUNTIF('アーク訪看'!B8:B208,"書類選考中")
日程調整中 | =COUNTIF('アーク訪看'!B8:B208,"日程調整中")
...

[空行]

[月間推移セクション] ← 開始月〜当月のみ
月     | 人数
11月   | =SUMPRODUCT(...)
12月   | =SUMPRODUCT(...)
1月    | =SUMPRODUCT(...)

[空行]

[職種セクション]
職種    | 人数
看護師   | =COUNTIF('アーク訪看'!J8:J208,"看護師")
...
```

#### 月間推移の数式

日付がテキスト `M/D` 形式とDateシリアル値の両方に対応:

```
=SUMPRODUCT(
  (ISNUMBER('ソース'!E8:E208)*(TEXT('ソース'!E8:E208,"M")="11")
  +(NOT(ISNUMBER('ソース'!E8:E208))*(LEFT('ソース'!E8:E208,LEN("11/"))="11/"))
  )*1
)
```

#### 月間推移の範囲

**開始月〜当月のみ表示**する。12ヶ月全部並べない。

1. ソースの日付列から最小月を特定
2. 当月（現在月）を終了月とする
3. 年跨ぎを考慮して月リストを生成（例: 10月→11月→12月→1月→2月→3月）

### 4. ダッシュボードシート作成

#### レイアウト

| エリア | 行 | 内容 |
|--------|-----|------|
| タイトルバー | 1 | シート名 + ダッシュボード（50px、ダーク背景、白文字20pt） |
| KPIラベル | 2 | 指標名（28px、白背景、ブランドカラー文字11pt） |
| KPI値 | 3 | 大きな数字（80px、白背景、42pt太字） |
| スペーサー | 4 | 背景色、21px |
| グラフ段1 | 5-19 | 左右2枚のグラフ |
| スペーサー | 20 | 背景色、8px |
| グラフ段2 | 21-35 | 左右2枚 |
| スペーサー | 36 | 背景色、8px |
| グラフ段3 | 37-51 | 左右2枚 |
| 背景 | 52-55 | 背景色で塗る |

**背景は55行目まで塗る。**

グラフ段の間にスペーサー行を入れて上下のグラフ間を空ける。

#### KPI数式

```
合計: =SUMPRODUCT((LEN('ソース'!A8:A208)>0)*1)
各ステータス: =COUNTIF('ソース'!B8:B208,"ステータス名")
```

### 5. addChart APIでグラフ作成

```python
{
    "addChart": {
        "chart": {
            "spec": {
                "title": "選考状況",
                "basicChart": {
                    "chartType": "COLUMN",  # or "BAR"
                    "legendPosition": "NO_LEGEND",
                    "axis": [
                        {"position": "BOTTOM_AXIS", "title": ""},
                        {"position": "LEFT_AXIS", "title": "人数"},
                    ],
                    "domains": [{
                        "domain": {
                            "sourceRange": {
                                "sources": [{
                                    "sheetId": chart_data_sheet_id,
                                    "startRowIndex": start, "endRowIndex": end,
                                    "startColumnIndex": 0, "endColumnIndex": 1,
                                }]
                            }
                        }
                    }],
                    "series": [{
                        "series": {
                            "sourceRange": {
                                "sources": [{
                                    "sheetId": chart_data_sheet_id,
                                    "startRowIndex": start, "endRowIndex": end,
                                    "startColumnIndex": 1, "endColumnIndex": 2,
                                }]
                            }
                        },
                        "color": {"red": R, "green": G, "blue": B},
                    }],
                    "headerCount": 1,
                },
            },
            "position": {
                "overlayPosition": {
                    "anchorCell": {
                        "sheetId": dashboard_sheet_id,
                        "rowIndex": row,
                        "columnIndex": col,
                    },
                    "widthPixels": 560,
                    "heightPixels": 320,
                }
            },
        }
    }
}
```

#### グラフ配置（2列 × 3段）

| 段 | 左グラフ | 右グラフ | anchorCell rowIndex |
|----|---------|---------|---------------------|
| 1 | 選考状況 | 月別推移 | 4 |
| 2 | 職種別 | 年齢分布 | 20 |
| 3 | 男女比 | 応募種別 | 36 |

左: columnIndex=0、右: columnIndex=6

### 6. フォーマット設定

- グリッドライン非表示
- 列幅: A〜L列を95px統一
- `_chart_data` シートは非表示にする

#### カラーパレット

| 役割 | 用途 |
|------|------|
| DARK | タイトルバー |
| MAIN | グラフメイン色、KPIラベル色 |
| MID | グラフ補助色 |
| LIGHT | グラフ薄色 |
| PALE | スペーサー行背景 |
| BG | シート背景（55行目まで） |

実績:
- いちご訪看: ティール/グリーン系
- アーク訪看: ブルー系
- LCC訪看: ウォーム/オレンジ系
- an訪看: パープル系

## 必須ルール

### 数値フォーマット

| 種類 | フォーマット |
|------|-------------|
| 人数・件数 | 整数 `#,##0` |
| 割合・進捗率 | パーセント `0%` or `0.0%` |

### 0-indexed行番号

Sheets APIの startRowIndex は0-indexed。スプレッドシートの行番号から -1 する。

### GAS不要

addChart方式では `_chart_data` の数式がソースを直接参照するため、GAS同期は不要。
`dashboard_sync.gs` のトリガーは削除すること。

## 検証チェックリスト

1. KPIの値とフォーマット（整数/%）
2. 全6グラフが描画されている
3. グラフの色がブランドカラーで統一
4. 月間推移が開始月〜当月のみ表示されている（未来月がない）
5. `_chart_data` シートが非表示になっている
6. 背景が55行目まで塗られている
7. グラフ段の間にスペーサーがある
8. ソースデータの日付がテキスト形式で統一されている
9. ソースシートの職種列がお客様行動管理と一致している

## よくあるミス

| ミス | 原因 | 対策 |
|------|------|------|
| 月間推移に未来月が表示される | 12ヶ月固定で出力 | 開始月〜当月のみに絞る |
| 日付のCOUNTIFが合わない | テキスト/シリアル値混在 | 全セルをテキスト `M/D` に統一、`RAW` で書き込み |
| テーブル列にsetDataValidationが使えない | ソースがテーブル機能を使っている | テーブル列のDROPDOWN設定はSheets UIで手動設定 |
| KPIが空文字をカウントする | COUNTAが空文字""を数える | SUMPRODUCT((LEN(...)>0)*1) を使う |
| グラフ位置が重なる | スペーサー行なし | グラフ段の間にスペーサー行を入れる |
| addChartのシリーズ色が効かない | colorフィールドの形式ミス | `{"red": 0.2, "green": 0.5, "blue": 0.8}` のfloat形式 |
