# 良い例保存スキル

生成したスカウト文の中から、特に出来が良いものや参考にしたいものを会社別の `history/examples.md` に蓄積します。

## 実行手順

1. **引数の解析**
   - 会社名、特徴（タイトル）、パーソナライズ文を取得します。
   - 引数が不足している場合は対話形式で確認します。

2. **保存先の特定**
   - `companies/[会社名]/history/examples.md` をターゲットにします。
   - ファイルが存在しない場合は、標準的なフォーマットで新規作成します。

3. **内容の追記**
   - 現在の年月（YYYY年MM月）のセクション配下に追記します。
   - フォーマット:
     ```markdown
     ### [特徴/タイトル]
     > [パーソナライズ文]
     ```

4. **完了報告**
   - 保存完了のメッセージと、保存した内容を表示します。

## 使用例

```
/save-example ark-visiting-nurse 10年以上のベテラン向け プロフィールを拝見し、10年以上にわたり...
```
