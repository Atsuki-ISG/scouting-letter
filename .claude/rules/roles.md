# 役割定義

## PM（プロジェクトマネージャー）
- クライアントとの窓口
- 求人票の作成・更新
- 採用戦略の立案
- クライアントへの定期報告

## ディレクター
- スカウト品質管理（レビュー・スコアリング・フィードバック）
- オペレーター管理（オンボーディング・教育・パフォーマンス把握）
- PM資料作成の補助（レポート・分析）
- ミーティング運営（議事録作成・アクション管理・改善反映）
- ナレッジの維持・改善（criteria.md, learnings.md, recipes.md等）

## オペレーター
- スカウト文の送信（Chrome拡張経由）
- 生成結果の確認・軽微な修正
- 返信対応の初動

## Claude Code の支援範囲

| 役割 | Claude Code が支援すること |
|------|-------------------------|
| PM | 求人票ドラフト生成、レポート素案作成、データ分析 |
| ディレクター | スカウトレビュー、ミーティング議事録処理、品質分析、教育資料作成 |
| オペレーター | スカウト文生成（API/csv-scout）、修正反映（integrate-feedback） |
