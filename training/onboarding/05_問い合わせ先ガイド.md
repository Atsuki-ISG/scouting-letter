# 問い合わせ先ガイド

困ったときの連絡先と、誰に聞けばいいかの振り分けです。

---

## 連絡先

| 担当 | 名前 | 担当範囲 |
|------|------|---------|
| 開発者・ツール担当 | あっきー | 拡張機能の操作方法・エラー・設定など、ツールやシステムに関すること全般 |
| PM（プロジェクトマネージャー） | ひろちゃん | どの会社に送るか・スカウト文の内容・候補者の判断など、企業個別の業務に関すること |

---

## あっきー（開発者・ツール担当）に聞くこと

ツールの操作方法やシステムの問題は、あっきーに連絡してください。

### 操作方法がわからないとき

| こんなとき | 具体例 |
|-----------|--------|
| **拡張機能のインストール** | 「Chromeへの登録がうまくいかない」「distフォルダが見つからない」 |
| **初期設定** | 「API設定のやり方がわからない」「接続テストの方法は？」 |
| **抽出の操作** | 「抽出件数はどこで変える？」「開始会員番号って何？」 |
| **生成の操作** | 「職種の選び方がわからない」「自動判定って何？」 |
| **送信の操作** | 「連続送信の始め方は？」「手動モードへの切り替え方は？」 |
| **求人自動選択** | 「自動選択をOFFにする方法は？」「手動で求人を選ぶには？」 |
| **ドライランモード** | 「テスト送信のやり方は？」 |
| **CSVインポート** | 「外部で作ったCSVの読み込み方は？」 |

### エラー・不具合が起きたとき

| こんなとき | 具体例 |
|-----------|--------|
| **接続テストが失敗する** | 「URL・パスワードは合ってるのに繋がらない」 |
| **サーバーエラー** | 「タイムアウトが連続する」「503エラーが出る」 |
| **抽出が動かない** | 「プロフィール画面が開かない」「途中で止まる」 |
| **抽出データがおかしい** | 「全員の資格が『不明』になる」 |
| **生成が全部エラー** | 「一括生成を押してもエラーしか出ない」 |
| **スカウト文が入力されない** | 「OKを押したのに文章がテキスト欄に入らない」 |
| **求人自動選択が動かない** | 「全員で求人選択が失敗する」 |
| **画面の表示がおかしい** | 「サイドパネルが真っ白」「ボタンが表示されない」 |
| **拡張機能のアップデート** | 「新しいバージョンの入れ方がわからない」 |

### 設定関連

| こんなとき | 具体例 |
|-----------|--------|
| **API設定情報** | 「エンドポイントURLやパスワードを教えてほしい」 |
| **拡張機能ファイル** | 「最新のZIPをもらいたい」 |
| **管理画面の操作** | 「求人を追加・変更したい」「テンプレートを修正したい」 |

## ひろちゃん（PM）に聞くこと

企業ごとの業務内容や判断に関することは、ひろちゃんに連絡してください。

### 業務指示・方針

| こんなとき | 具体例 |
|-----------|--------|
| **どの会社にスカウトを送るか** | 「今日はどの会社の分をやればいいですか？」 |
| **送信件数・ペースの目標** | 「今日は何件くらい送ればいいですか？」 |
| **業務の優先順位** | 「初回と再送、どっちを先にやる？」 |
| **シフト・稼働時間の相談** | 「今週は水曜に休みたい」 |

### 企業個別の判断

| こんなとき | 具体例 |
|-----------|--------|
| **候補者の絞り込み条件** | 「この会社はどのエリアの候補者を対象にすればいい？」 |
| **職種の選び方** | 「この候補者は看護師と保健師の両方持ってるけど、どっちで送る？」 |
| **送信対象の判断** | 「黄色の警告が出てるけど、この人には送っていい？」 |
| **スカウト文の内容** | 「この文章、この会社の特徴と合ってる？」 |
| **再送の判断** | 「この会社は再送もやったほうがいい？」 |
| **イレギュラー対応** | 「特定の候補者を優先してほしいと言われた」 |

### アカウント関連

| こんなとき | 具体例 |
|-----------|--------|
| **ジョブメドレーのログイン** | 「ログインできない」「パスワードを忘れた」 |
| **アカウントの権限** | 「この事業所のページが見られない」 |

## 迷ったときの判断基準

```
ツールの操作・設定・エラー → あっきー
企業の業務内容・判断 → ひろちゃん
どちらかわからない → ひろちゃん（必要ならあっきーに繋いでくれます）
```

もう少し具体的に：

- 「ボタンを押しても動かない」→ あっきー
- 「ボタンは動くけど、この候補者に送っていいかわからない」→ ひろちゃん
- 「エラーメッセージが出る」→ あっきー
- 「送る会社や職種がわからない」→ ひろちゃん

---

## 連絡するときに伝えてほしいこと

### あっきーへの連絡時

スムーズに原因を特定するために、以下をできるだけ添えてください：

1. **スクリーンショット**（エラー画面を撮る。これが一番助かります）
2. 何をしようとしていたか（どのタブで何のボタンを押したか）
3. いつから起きているか（突然か、最初からか）
4. 再試行したか、その結果はどうだったか

### ひろちゃんへの連絡時

1. 何をしようとしていたか
2. 何に迷っているか・判断がつかないポイント
3. 該当する候補者の情報（会員番号など）があれば
