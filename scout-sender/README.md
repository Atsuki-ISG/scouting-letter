# Scout Sender - プロフィール一括抽出

ジョブメドレーのスカウト対象者一覧ページから、プロフィール情報を一括抽出してCSVファイルとしてダウンロードするChrome拡張機能です。

## できること

- スカウト対象者一覧から指定件数のプロフィールを自動抽出
- 開始会員番号を指定して途中から抽出再開
- 抽出結果をCSVファイルとしてダウンロード

## 抽出される項目

会員番号、性別、年齢、居住地、資格、経験職種、経験年数、就業状況、希望職種、希望勤務地、希望勤務形態、希望入職時期、自己PR、こだわり条件、職務経歴

## セットアップ

### 1. ビルド

```bash
cd scout-sender
npm install
npm run build
```

`dist/` フォルダにビルド済みファイルが出力されます。

### 2. Chromeにインストール

1. Chromeで `chrome://extensions/` を開く
2. 右上の **「デベロッパーモード」をON** にする
3. **「パッケージ化されていない拡張機能を読み込む」** をクリック
4. `scout-sender/dist` フォルダを選択

### 3. 使い方

1. ジョブメドレーの管理画面で **スカウト対象者の一覧ページ** を開く
2. ツールバーの拡張機能アイコンをクリック → サイドパネルが開く
3. **抽出件数** を入力（デフォルト5件、最大50件）
4. 途中から始めたい場合は **開始会員番号** を入力
5. **「抽出開始」** をクリック
6. 自動的にスカウト画面を開閉しながらプロフィールを取得
7. 完了後 **「CSVダウンロード」** をクリック

### 注意事項

- 抽出中はページを操作しないでください（overlayの自動開閉が進行中のため）
- 1ページに表示されている件数以上は抽出できません（次ページへは手動で移動してください）
- 抽出途中で中断した場合も、それまでの結果はCSVダウンロード可能です

## 開発者向け

### ビルド

```bash
npm run build
```

### 構成

```
scout-sender/
├── src/
│   ├── shared/          # 型定義・ストレージ・CSV・定数
│   ├── content/         # Content Script（DOM操作・プロフィール抽出）
│   ├── background/      # Service Worker（メッセージルーティング）
│   └── sidepanel/       # サイドパネルUI
├── scripts/build.js     # ビルドスクリプト
├── manifest.json
└── dist/                # ビルド出力（Chromeにはこれを読み込む）
```
