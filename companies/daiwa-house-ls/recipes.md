# パーソナライズ レシピ: ネオ・サミット湯河原（入居相談員）

## 当施設の特色（パーソナライズで使える素材）

- 大和ハウスグループの安定基盤、充実した福利厚生
- 有料老人ホームの入居募集営業（紹介会社・病院・介護事業者・役所等へのアプローチ）
- 見学会の企画・案内、入居契約対応
- 年間休日121日、ワークライフバランス重視
- OJT・メンター制度あり、中途入社でも馴染みやすい環境
- 営業奨励金制度あり（成果に応じた報酬）
- 「ご入居者お一人おひとりの生き方に寄り添う」理念

## 共通ルール

- 書き出し: テンプレート冒頭に「ご経歴を拝見し」があるため「拝見し」以外で開始する
- 文字数: 初回 150〜200文字 / 再送 150〜200文字
- トーン: 丁寧で誠実だが堅すぎない。大和ハウスグループの信頼感を保つ
- 年齢・世代への言及禁止（「〇代」「お若さ」「ベテラン」等）
- 居住地に言及する場合は広域表現に留める（「〇〇区」→「静岡県内」等）
- 経験年数の記載ルール:
  - 1〜2年: 書かない
  - 3年以上: 書く
- 堅苦しい敬語禁止（「拝察いたします」「敬意を表します」等）
- 上から目線の排除（「安心してスタートいただけます」等）
- 憶測を書かない（「〜されたいのですね」「長く続けているはず」）
- ですます調で統一

## パーソナライズの基本方針

入居相談員は**営業職**であり、介護・医療の現場経験を以下の観点で転用する:

| 候補者の経験 | 接点の作り方 |
|---|---|
| 介護職/ヘルパー | 現場で培った入居者理解・ご家族対応力が相談業務に直結 |
| 生活相談員 | 入居相談・契約対応の経験がそのまま活きる |
| 管理職（介護） | マネジメント経験＋多職種連携力が営業活動の強み |
| 介護事務 | 事務処理能力＋介護業界の知識が見学案内・契約対応に活きる |
| ケアマネジャー | ケアプラン作成で培った提案力・調整力が営業に転用できる |
| 営業職 | 営業スキルがそのまま活きる、介護業界での営業は社会貢献性が高い |
| 看護師 | 医療的な視点が入居者・ご家族への説明力に直結 |

## 型はめパターン

情報が少ない候補者（経歴・自己PRなし）に使用する。
特色バリエーションは候補者ごとにローテーションし、同じ文の連続を防ぐ。

### 型A: 豊富な介護経験からの転身

対象: 介護職/ヘルパー経験10年+ / 管理職経験あり

```
{N}年以上にわたり介護の現場でご活躍されてきたご経歴に、大変注目いたしました。現場で培われた入居者様への深い理解やご家族との信頼関係の築き方は、{特色}入居相談員として大きな強みになると考えております。
```

特色バリエーション:
- 入居をご検討されるご家族に寄り添った提案が求められる
- 紹介会社や医療機関との連携が重要となる
- 見学会の企画やご契約対応など幅広い業務を担う

### 型B: 確かな介護経験×営業適性

対象: 介護職/ヘルパー経験3〜9年

```
介護職として{N}年にわたり現場で培われたご経験やスキルは、{特色}入居相談員の業務においても大いに活かしていただけると考えております。
```

特色バリエーション:
- 入居者様やご家族のニーズを深く理解した提案が求められる
- 介護事業者や病院等との連携・アプローチを行う
- 見学会のご案内やご契約対応を通じてご入居者の暮らしを支える

### 型C: 生活相談員・ケアマネ経験

対象: 生活相談員/ケアマネジャー経験あり

```
生活相談員としてのご経験で培われた、ご入居者やご家族への相談対応力・関係機関との調整力は、{特色}当施設の入居相談員として即戦力となるスキルだと考えております。
```

特色バリエーション:
- 入居をご検討されるご家族への丁寧な提案が求められる
- 紹介会社・病院・介護事業者等への営業アプローチを行う
- ご入居者お一人おひとりの生き方に寄り添うサービスを提供する

### 型D: 介護事務経験

対象: 介護事務経験あり

```
介護事務としてのご経験で培われた事務処理能力と介護業界への深い理解は、{特色}入居相談員の業務において大きなアドバンテージになると考えております。
```

特色バリエーション:
- 資料作成や契約対応など正確さが求められる業務が多い
- 介護業界の知識を活かした見学案内やご家族への説明が求められる
- 紹介会社や医療機関との連携を円滑に進める

### 型E: 営業経験からの転身

対象: 営業職経験あり（業種問わず）

```
営業職としてのご経験で培われた提案力やコミュニケーション力は、{特色}入居相談員の業務において即戦力として発揮していただけると考えております。
```

特色バリエーション:
- 紹介会社・病院・役所等への法人営業を中心に行う
- 見学会の企画・案内からご契約までを一貫して担当する
- ご入居者お一人おひとりの生き方に寄り添う営業活動を行う

### 型F: 看護師経験からの転身

対象: 看護師/准看護師経験あり

```
看護師として医療の現場で培われた知識やご経験は、{特色}入居相談員としてご入居者やご家族に安心感のあるご提案をしていただくうえで、大きな強みになると考えております。
```

特色バリエーション:
- 医療的なケアが必要なご入居者のご家族への説明が求められる
- 病院や介護事業者との連携・アプローチを行う
- ご入居者の健康状態を踏まえた入居相談に対応する

### 型G: 経験少ない／未入力

対象: 介護・医療系経験1〜2年 / 経験未入力だが関連資格あり

就業中:
```
現在も介護の現場でご活躍されているご経験は、{特色}当施設の入居相談員として活かしていただける場面が多いと考えております。当社が目指す「ご入居者お一人おひとりの生き方に寄り添う」という理念の実現において、現場を知る方の力は非常に重要です。
```

離職中:
```
これまで介護の現場で培われてきたご経験は、{特色}当施設の入居相談員として活かしていただける場面が多いと考えております。当社が目指す「ご入居者お一人おひとりの生き方に寄り添う」という理念の実現において、現場を知る方の力は非常に重要です。
```

特色バリエーション:
- 入居者様やご家族のニーズを理解した相談対応が求められる
- 見学会のご案内やご契約対応を通じてご入居者の暮らしを支える
- 紹介会社や医療機関との連携を担う

## AI生成ガイド（経歴・自己PRがある候補者向け）

経歴や自己PRが充実している候補者には、型はめではなくAI生成を使う。

### 生成指示

以下の観点で150〜200文字のパーソナライズ文を生成する:

1. **経験の転用**: 候補者の介護・医療・営業経験が入居相談員としてどう活きるかを具体的に示す
2. **接点の明示**: 候補者の経験と当施設の業務内容（営業・見学案内・契約対応）の接点を1つ選ぶ
3. **理念との結びつき**: 「ご入居者お一人おひとりの生き方に寄り添う」理念と候補者の経験を自然に結びつける

### 禁止事項

- 年齢・世代への言及
- 「お持ちとのこと」等の回りくどい敬語
- 送り手の感情主体（「ご一緒したい」「想いを強く感じ」）
- 居住地への詳細な言及
- テンプレートと重複する内容（募集要項・福利厚生の説明等）
