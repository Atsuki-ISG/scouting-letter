# パーソナライズ レシピ: ネオ・サミット湯河原（入居相談員）

## 当施設の特色（パーソナライズで使える素材）

- 大和ハウスグループの安定基盤、充実した福利厚生
- 有料老人ホームの入居募集営業（紹介会社・病院・介護事業者・役所等へのアプローチ）
- 見学会の企画・案内、入居契約対応
- 年間休日121日、ワークライフバランス重視
- OJT・メンター制度あり、中途入社でも馴染みやすい環境
- 営業奨励金制度あり（成果に応じた報酬）
- 「ご入居者お一人おひとりの生き方に寄り添う」理念

## 共通ルール

- 書き出し: 「プロフィール/ご経歴を拝見し」は使わない（テンプレ冒頭の「ご経歴を拝見し」と重複するため）
- 文字数: 初回 約150〜200文字 / 再送 150〜200文字
- トーン: お手紙の感覚で丁寧に。観察・断定（「〜と考えております」）で閉じず、招待調（「ぜひ一度お話しできましたら嬉しく思います」）で締める
- 「当施設」ではなく「私たちの施設」で同じ地平に立つ（入居相談員の文脈では「当施設の入居相談員」と業務を指す場合のみ「当施設」可）
- 「大変心強く拝見しました」のような装飾副詞付き観察調は避ける。「強く惹かれました」「感じています」等の控えめな一段反応で
- 年齢・世代への言及禁止（「〇代」「お若さ」「ベテラン」等）
- 居住地に言及する場合は広域表現に留める（「〇〇区」→「静岡県内」等）
- 経験年数の記載ルール:
  - 1〜2年: 書かない
  - 3年以上: 書く
- 「感銘を受けました」「心を動かされました」は使わない（AI生成だと白々しい）
- 堅苦しい敬語禁止（「拝察いたします」「敬意を表します」等）
- 上から目線の排除（「安心してスタートいただけます」等）
- 送り手の感情主体（「ご一緒したい」「想いを強く感じ」）は避け、相手へのオファー主体で
- 憶測を書かない（「〜されたいのですね」「長く続けているはず」）
- ですます調で統一

## パーソナライズの基本方針

入居相談員は営業職であり、介護・医療の現場経験を以下の観点で転用する:

| 候補者の経験 | 接点の作り方 |
|---|---|
| 介護職/ヘルパー | 現場で培った入居者理解・ご家族対応力が相談業務に直結 |
| 生活相談員 | 入居相談・契約対応の経験がそのまま活きる |
| 管理職（介護） | マネジメント経験＋多職種連携力が営業活動の強み |
| 介護事務 | 事務処理能力＋介護業界の知識が見学案内・契約対応に活きる |
| ケアマネジャー | ケアプラン作成で培った提案力・調整力が営業に転用できる |
| 営業職 | 営業スキルがそのまま活きる、介護業界での営業は社会貢献性が高い |
| 看護師 | 医療的な視点が入居者・ご家族への説明力に直結 |

## 型はめパターン

情報が少ない候補者（経歴・自己PRなし）に使用する。
特色バリエーションは候補者ごとにローテーションし、同じ文の連続を防ぐ。

### 型A: 豊富な経験への期待

対象: 経験10年+ / 40代〜×経験6年+

```
10年以上にわたり介護・福祉の現場で歩んでこられたご経歴に惹かれました。ご入居者様やご家族と真摯に向き合ってこられたご経験は、{特色}私たちの入居相談員のお仕事で、自然と活きる場面が多いのではと感じています。ぜひ一度お話しできましたら嬉しく思います。
```

特色バリエーション:
- ご入居を検討されるご本人様やご家族の気持ちに寄り添う
- ご入居者お一人おひとりの生き方に寄り添うサービスを提供する
- 見学会のご案内や入居のご相談対応を通じて、新しい暮らしの始まりを支える

### 型B1: 確かな経験×特色

## AI生成ガイド（経歴・自己PRがある候補者向け）

経歴や自己PRが充実している候補者には、型はめではなくAI生成を使う。

### 生成指示

以下の観点で150〜200文字のパーソナライズ文を生成する:

1. 経験の転用: 候補者の介護・医療・福祉・営業の経験が、入居相談員という仕事でどう活きるかを具体的に1つ挙げる
2. 業務の人間味: 入居相談員は「人生の新しい段階を迎えるご入居者ご本人様やご家族の、大きな決断に寄り添う」仕事である。この視点から候補者の経験の意味を言語化する
3. 理念との結びつき: 「ご入居者お一人おひとりの生き方に寄り添う」理念と候補者の経験を自然に繋ぐ
4. 締め: 「〜と感じています。ぜひ一度お話しできましたら嬉しく思います」で招待調に閉じる

### 禁止事項（お手紙トーンを損なうため避ける）

- 年齢・世代への言及
- 「お持ちとのこと」等の回りくどい敬語
- 送り手の感情主体（「ご一緒したい」「想いを強く感じ」）
- 居住地への詳細な言及
- テンプレートと重複する内容（募集要項・福利厚生の説明等）
- **営業・管理色の強い表現**: 「施設の顔」「営業としての即戦力」「管理職へのステップアップ」「10年、20年先を見据えたキャリア」等の営業・昇進を前面に出す書き方は避ける。候補者の経験を、ご入居者ご家族の気持ちに寄り添う相談員という仕事に接続する
- **「営業」という言葉自体を使わない**: 業務は「入居を検討されるご入居者ご家族へのご案内・ご相談対応」など、相談員の仕事として描写する
