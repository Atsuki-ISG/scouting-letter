# パーソナライズ レシピ: アーク訪問看護ステーション

## 当ステーションの特色（パーソナライズで使える素材）

- 認知症の利用者様が8割（認知症ケアの経験を活かせる）
- クリニック併設で医師と密に連携できる（病棟経験のチーム連携力が活きる）
- 介護予防から在宅看取りまで幅広く対応（幅広い経験を活かせる）
- 在宅看取り年間約20件（ターミナルケア経験が活きる）
- 利用者様一人ひとりと深く向き合う看護（対話力・関係性構築力が活きる）

## 共通ルール

- 書き出し: 「プロフィール/ご経歴を拝見し」は使わない（テンプレ冒頭の「ご経歴を拝見し」と重複するため）
- 文字数: 初回 約100〜180文字 / 再送 120〜180文字
- トーン: お手紙の感覚で丁寧に。観察・断定（「〜と考えております」）で閉じず、招待調（「ぜひ一度お話しできましたら嬉しく思います」）で締める
- 「当ステーション」ではなく「私たちの訪問看護」で同じ地平に立つ
- 「大変心強く拝見しました」のような装飾副詞付き観察調は避ける。「強く惹かれました」「感じています」等の控えめな一段反応で
- 地理情報（勤務地・訪問範囲）はテンプレートに記載済みのため言及しない
- 年齢・世代への言及禁止（「〇代」「若手」「ベテラン」等）
- 経験年数の記載ルール:
  - 1〜2年: 書かない
  - 3年: 30代前半以下なら書く、30代後半以上なら書かない
  - 4年以上: 書く
- 「感銘を受けました」「心を動かされました」は使わない（AI生成だと白々しい）
- こだわり条件は参考程度。資格・経験で接点が作れるなら言及不要
- 上から目線の排除（「フォローします」「安心してください」「基礎を固めてこられた」）
- 送り手の感情主体（「ご一緒したい」「想いを強く感じ」）は避け、相手へのオファー主体で
- ですます調で統一。体言止めや「〜とのこと、」で文を切らない
- 憶測を書かない（「〜されたいのですね」「長く続けているはず」）
- 不要な前置きを書かない（「現在離職中とのこと」「新たな環境でとお考えの〜」）
- 居住地に言及する場合は広域表現に留める（「〇〇区」→「札幌市内」）
- 堅苦しい敬語禁止（「拝察いたします」「敬意を表します」等）

## 教育体制（当ステーション固有 — 他社の制度を混ぜないこと）

- S-QUE訪問看護eラーニング
- 入社初日からの新任研修制度
- セミナー・研修の費用補助（年間50,000円）

## 型はめパターン

情報が少ない候補者（経歴・自己PRなし）に使用する。
特色バリエーションは候補者ごとにローテーションし、同じ文の連続を防ぐ。

### 型A: 豊富な経験への期待

対象: 経験10年+ / 40代〜×経験6年+

```
10年以上にわたり看護師としてご活躍されてきたご経歴に、強く惹かれました。豊富な臨床経験で培われた確かな技術とアセスメント力は、{特色}私たちの訪問看護で大きな力になっていただけると感じています。ぜひ一度お話しできましたら嬉しく思います。

- 利用者様一人ひとりと深く向き合う
- 認知症の利用者様が多く、きめ細かなケアが求められる
- 介護予防から在宅看取りまで幅広く対応する

### 型B1: 確かな経験×特色

```

## 資格修飾（複数資格がある場合、型の冒頭を差し替え）

型はめパターンの書き出し部分を以下に差し替える。後半（特色部分）はそのまま使う。

### 看護師 + 保健師

```
看護師と保健師の両資格をお持ちで、保健師として培われた予防的視点やアセスメント力は、在宅で利用者様の生活全体を支える訪問看護において大きな強みになると考えております。
```

### 看護師 + ケアマネージャー

```
看護師の資格に加え、ケアマネージャーの資格もお持ちとのこと、医療と介護の両面から利用者様を支える視点は、多職種と連携しながら訪問看護に取り組む当ステーションで大きな力になると考えております。
```

### 看護師 + 認定看護師等の専門資格

```
{資格名}の資格をお持ちで、その専門性は{関連する特色}当ステーションにおいて、大きな力になると考えております。
```

## AI生成パス用ガイド

経歴・自己PRに実質的な情報がある候補者に使用する。

### 生成手順

1. 候補者の経歴・自己PRから最も強い接点を1つ選ぶ
   - 経験年数が4年以上の候補者には、パーソナライズ文中に具体的な経験年数を含めること（例: 「10年以上の臨床経験」「5年にわたる〜」）
2. 上記「当ステーションの特色」から対応する強みを選ぶ
3. 接点を100文字程度で表現する

### 経歴別の接点対応表

| 候補者の経験 | 対応するステーション特色 | 接点の方向性 |
|------------|----------------------|------------|
| 認知症ケア | 利用者の8割が認知症 | 「認知症ケアのご経験は、当ステーションで大きな力になる」 |
| 緩和ケア・ターミナル | 在宅看取り年間約20件 | 「看取りに向き合ってこられた経験は、当ステーションが大切にしている姿勢と重なる」 |
| 病棟看護（急性期等） | クリニック併設・医師連携 | 「病棟でのチーム連携のご経験は、クリニック併設の当ステーションで活かせる」 |
| 訪問看護経験 | 地域密着・対話重視 | 「訪問看護で培われた対話力は、患者様との関係性作りにおいて大きな強みになる」 |
| 複数診療科の経験 | 幅広い疾患に対応 | 「幅広い臨床経験は、多様な利用者様に対応する当ステーションで活かせる」 |

### NGパターン

- 運転免許など弱い接点で文を構成する
- 会社情報を羅列するだけで接点になっていない
- 地理的要素のみに頼る
- 候補者の希望職種が看護師以外でも、看護師としての接点で書く（希望職種には言及しない）
