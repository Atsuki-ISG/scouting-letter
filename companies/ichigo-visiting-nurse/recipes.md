# パーソナライズ レシピ: いちご訪問看護リハビリステーション

## 当ステーションの特色（パーソナライズで使える素材）

- 介護保険の訪問がメイン、医療依存度の高い利用者は数名（幅広い経験を活かせる）
- 看護師とリハビリ職が5:5、多職種連携で利用者様を支える
- 練馬区・杉並区の一部で地域密着の訪問看護
- 「一期一会を大切に、ご利用様に寄り添ったサービスを提供する」理念
- 事業拡大中（東京・愛知に拠点）、ステーション立ち上げやマネジメントを学べる環境
- ターミナル加算・24時間対応体制加算あり

## 共通ルール

- 書き出し: 「プロフィール/ご経歴を拝見し」は使わない（テンプレ冒頭の「ご経歴を拝見し」と重複するため）
- 文字数: 初回 約100〜180文字 / 再送 120〜180文字
- トーン: お手紙の感覚で丁寧に。観察・断定（「〜と考えています」）で閉じず、招待調（「ぜひ一度お話しできましたら嬉しく思います」）で締める
- 「当ステーション」ではなく「私たちの訪問看護」で同じ地平に立つ
- 「大変心強く拝見しました」のような装飾副詞付き観察調は避ける。「強く惹かれました」「感じています」等の控えめな一段反応で
- 地理情報（勤務地・訪問範囲）はテンプレートに記載済みのため言及しない
- 年齢・世代への言及禁止（「〇代」「若手」「ベテラン」等）
- 経験年数の記載ルール:
  - 1〜2年: 書かない
  - 3年: 30代前半以下なら書く、30代後半以上なら書かない
  - 4年以上: 書く
- 「感銘を受けました」「心を動かされました」は使わない（AI生成だと白々しい）
- こだわり条件は参考程度。資格・経験で接点が作れるなら言及不要
- 上から目線の排除（「フォローします」「安心してください」「基礎を固めてこられた」）
- 送り手の感情主体（「ご一緒したい」「想いを強く感じ」）は避け、相手へのオファー主体で
- ですます調で統一。体言止めや「〜とのこと、」で文を切らない
- 憶測を書かない（「〜されたいのですね」「長く続けているはず」）
- 不要な前置きを書かない（「現在離職中とのこと」「新たな環境でとお考えの〜」）
- 居住地に言及する場合は広域表現に留める（「〇〇区」→「都内」）
- 堅苦しい敬語禁止（「拝察いたします」「敬意を表します」等）

## 教育体制（当ステーション固有 — 他社の制度を混ぜないこと）

- 同行訪問・オンライン研修あり
- 資格取得支援
- ジョブメドレーアカデミー利用

## 型はめパターン

情報が少ない候補者（経歴・自己PRなし）に使用する。
特色バリエーションは候補者ごとにローテーションし、同じ文の連続を防ぐ。

### 型A: 豊富な経験への期待

対象: 経験10年+ / 40代〜×経験6年+

```
10年以上にわたり看護師としてご活躍されてきたご経歴に、強く惹かれました。豊富な臨床経験で培われた確かな技術とアセスメント力は、{特色}私たちの訪問看護で大きな力になっていただけると感じています。ぜひ一度お話しできましたら嬉しく思います。
```

特色バリエーション:
- 利用者様一人ひとりに寄り添うケアを大切にする
- 看護師とリハビリ職が連携して利用者様を支える
- 介護保険の利用者様を中心に地域に根差したケアを提供する

### 型B1: 確かな経験×特色

対象: 経験6〜9年

```
{N}年にわたり看護師として培われた確かなご経験とスキルは、{特色}私たちの訪問看護で存分に活かしていただけると感じています。ぜひ一度お話しできましたら嬉しく思います。
```

特色バリエーション:
- 看護師とリハビリ職が5:5で連携しながら訪問看護に取り組む
- 介護保険中心に利用者様の生活に寄り添う
- 事業拡大中でキャリアの幅を広げられる環境の

### 型B2: 経験×特色

対象: 経験3〜5年

```
看護師として{N}年にわたり現場で培われたスキルは、{特色}私たちの訪問看護で活きる場面が多いと感じています。ぜひ一度お話しできましたら嬉しく思います。
```

特色バリエーション:
- 多職種が連携しチームで利用者様を支える
- 介護保険の利用者様を中心に地域に根差したケアを提供する
- 利用者様一人ひとりの暮らしに寄り添う

### 型C: 経験とのフィット

対象: 40代〜 × 経験1〜2年

```
{特色}私たちの訪問看護で、これまでの看護師としてのご経験をぜひ活かしていただきたいと思い、ご連絡いたしました。一度お話しできましたら嬉しく思います。
```

特色バリエーション:
- 介護保険の利用者様を中心に、一人ひとりに寄り添うケアを大切にする
- 看護師とリハビリ職が連携して利用者様を支える
- 地域に根差したケアを提供する

### 型D: 経験ある前提で評価

## 資格修飾（複数資格がある場合、型の冒頭を差し替え）

型はめパターンの書き出し部分を以下に差し替える。後半（特色部分）はそのまま使う。

### 看護師 + 保健師

```
看護師と保健師の両資格をお持ちで、保健師として培われた予防的視点やアセスメント力は、在宅で利用者様の生活全体を支える訪問看護において大きな強みになると感じています。
```

### 看護師 + ケアマネージャー

```
看護師の資格に加え、ケアマネージャーの資格もお持ちとのこと、医療と介護の両面から利用者様を支える視点は、多職種と連携しながら訪問看護に取り組む私たちの現場で大きな力になると感じています。
```

### 看護師 + 認定看護師等の専門資格

```
{資格名}の資格をお持ちで、その専門性は{関連する特色}私たちの訪問看護において、大きな力になると感じています。
```

## AI生成パス用ガイド

経歴・自己PRに実質的な情報がある候補者に使用する。

### 生成手順

1. 候補者の経歴・自己PRから最も強い接点を1つ選ぶ
   - 経験年数が4年以上の候補者には、パーソナライズ文中に具体的な経験年数を含めること（例: 「10年以上の臨床経験」「5年にわたる〜」）
2. 上記「当ステーションの特色」から対応する強みを選ぶ
3. 接点を100文字程度で表現する

### 経歴別の接点対応表

| 候補者の経験 | 対応するステーション特色 | 接点の方向性 |
|------------|----------------------|------------|
| 認知症ケア | 介護保険メインの訪問 | 「認知症ケアのご経験は、介護保険の利用者様が中心の私たちの訪問看護で大きな力になる」 |
| 緩和ケア・ターミナル | ターミナル加算対応 | 「看取りに向き合ってこられた経験は、私たちが大切にしている姿勢と重なる」 |
| 病棟看護（急性期等） | 多職種連携（看護×リハ5:5） | 「病棟でのチーム連携のご経験は、看護師とリハビリ職が密に連携する私たちの訪問看護で活かせる」 |
| 訪問看護経験 | 一期一会の理念・地域密着 | 「訪問看護で培われた対話力は、利用者様との関係性作りにおいて大きな強みになる」 |
| 複数診療科の経験 | 幅広い疾患に対応 | 「幅広い臨床経験は、多様な利用者様に対応する私たちの訪問看護で活かせる」 |
| 小児看護 | 小児にも対応 | 「小児看護のご経験は、幅広い利用者様に対応する私たちの訪問看護で」 |
| 精神科看護 | 精神疾患にも対応 | 「精神科での看護経験は、多様な利用者様に寄り添う私たちの訪問看護で」 |

### NGパターン

- 運転免許など弱い接点で文を構成する
- 会社情報を羅列するだけで接点になっていない
- 地理的要素のみに頼る
- 候補者の希望職種が看護師以外でも、看護師としての接点で書く（希望職種には言及しない）
