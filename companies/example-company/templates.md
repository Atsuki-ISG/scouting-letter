# テンプレート集: サンプル介護株式会社

## 使い方

これらはあくまでベースのテンプレートです。
必ず求職者のプロフィールに合わせてパーソナライズしてください。

---

## 介護福祉士向けテンプレート

### 件名例

- 「〇〇様｜介護福祉士としてのご経験を活かせる職場です」
- 「〇〇様へ、ユニットケアに力を入れる特養からのご案内」

### 本文テンプレート

```
〇〇様

はじめまして。サンプル介護株式会社の採用担当、山田と申します。

ジョブメドレーにて〇〇様のプロフィールを拝見し、
[経験年数]年の介護福祉士としてのご経験、
特に[具体的な経験・スキル]に携わってこられた点に魅力を感じ、
ご連絡させていただきました。

当施設「サンプル特別養護老人ホーム」は2019年開設、
10名1ユニットの少人数制で入居者様お一人おひとりに
寄り添ったケアを大切にしております。

タブレットでの記録やセンサー見守りシステムなどICTも導入しており、
残業は月平均5時間以下です。
〇〇様のこれまでのご経験を存分に活かしていただける環境です。

もしご興味をお持ちいただけましたら、
まずはオンラインでの施設説明からでもお気軽にご連絡ください。

〇〇様からのご返信を心よりお待ちしております。

サンプル介護株式会社
採用担当 山田花子
```

```
〇〇様

はじめまして。サンプル介護株式会社の採用担当、山田と申します。
```

```
ご希望とのことで、当社のデイサービス部門をご案内いたします。

残業はほとんどなく、有給消化率は85%。
お子様の行事やご家庭の用事にも柔軟に対応しております。

[求職者の経験に合わせた具体的なメリット]

まずは施設の雰囲気を見ていただくだけでも構いません。
ご興味がございましたらお気軽にご返信ください。

サンプル介護株式会社
採用担当 山田花子
```

## カスタマイズのポイント

### 必ず変更する箇所

- `〇〇様` → 求職者の名前
- `[経験年数]` → プロフィールから読み取った年数
- `[具体的な経験・スキル]` → プロフィールの特徴的な経験
- `[施設形態]` → 経験した施設の種類
- `[求職者の経験に合わせた具体的なメリット]` → マッチングポイント

### 状況に応じて追加

- 通勤に関する情報（エリアが近い場合）
- 給与に関する具体的な数字（希望年収がある場合）
- キャリアパスに関する情報（キャリアアップ志向の場合）
