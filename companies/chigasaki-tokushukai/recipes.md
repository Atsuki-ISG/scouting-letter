# パーソナライズ レシピ: 茅ヶ崎徳洲会病院

## 当院の特色（パーソナライズで使える素材）

- 急性期＋地域包括ケア病棟の2本柱（急性期から回復期まで幅広い経験が積める）
- 救急搬送1日6〜7件、落ち着いた急性期（忙しすぎない環境で質の高い看護）
- 新卒離職率10年連続0%の「人を大切に育てる風土」
- 中途入職者にも約3ヶ月間マンツーマンサポート
- 24時間託児所完備（0〜10歳、スポット利用可）、時短勤務充実
- 医療費減免制度（職員＋一等親家族、月3,000円超過分を病院が全額負担）
- 賞与3ヶ月分、残業月16〜20時間
- 徳洲会グループ（医療グループ日本一）の安定基盤
- 地域住民6,372名の署名で存続が決まった、地域に愛される病院

## 共通ルール

- 書き出し: テンプレート冒頭に「ご経歴と茅ヶ崎を希望勤務地に」があるため「拝見し」「ご経歴」以外で開始する
- 文字数: 約100文字
- トーン: 丁寧だが堅すぎない。病院らしい誠実さ＋親しみやすさ
- 地理情報はテンプレートに記載済みのため言及しない
- 年齢・世代への言及禁止（「〇代」「若手」「ベテラン」等）
- 経験年数の記載ルール:
  - 1〜2年: 書かない
  - 3年: 30代前半以下なら書く、30代後半以上なら書かない
  - 4年以上: 書く
- 「感銘を受ける」は経験そのものに使用禁止。代替:「注目しました」「心強く感じました」
- こだわり条件は参考程度。資格・経験で接点が作れるなら言及不要
- 上から目線の排除（「フォローします」「安心してください」「基礎を固めてこられた」）
- ですます調で統一。体言止めや「〜とのこと、」で文を切らない
- 憶測を書かない（「〜されたいのですね」「長く続けているはず」）
- 不要な前置きを書かない（「現在離職中とのこと」「新たな環境でとお考えの〜」）
- 堅苦しい敬語禁止（「拝察いたします」「敬意を表します」等）

## 教育体制（当院固有 — 他院の制度を混ぜないこと）

- 中途入職者に約3ヶ月間、専任担当者がマンツーマンでサポート
- eラーニングシステム（業務時間内受講可）
- 院内研修・勉強会
- 新卒離職率10年連続0%

## 型はめパターン

情報が少ない候補者（経歴・自己PRなし）に使用する。
特色バリエーションは候補者ごとにローテーションし、同じ文の連続を防ぐ。

### 型A: 豊富な経験への期待

対象: 経験10年+ / 40代〜×経験6年+

```
10年以上にわたり看護の現場でご活躍されてきたご経歴を、大変心強く拝見しました。豊富な臨床経験で培われた確かな技術とアセスメント力は、{特色}当院において大きな力になると考えています。
```

特色バリエーション:
- 急性期から地域包括ケアまで幅広い看護を提供する
- 救急搬送1日6〜7件の落ち着いた急性期環境で質の高い看護を目指す
- 多診療科の患者様に対応する
- 固定チームナーシングでスタッフ同士が支え合いながら看護に取り組む

### 型B1: 確かな経験×特色

対象: 経験6〜9年

```
看護師として{N}年にわたり臨床の現場で培われた確かな経験やスキルは、{特色}当院において大きな力になると考えています。
```

特色バリエーション:
- 急性期と地域包括ケアの2つの病棟で幅広い看護に取り組む
- チーム医療を大切にし、多職種が連携しながら患者様を支える
- 144床の地域密着型病院として多様な症例に対応する

### 型B2: 経験×特色

対象: 経験3〜5年

```
看護師として{N}年にわたり現場で培われたスキルは、{特色}当院において活かしていただけると考えています。
```

特色バリエーション:
- 急性期から回復期まで幅広い看護経験を積める環境の
- 固定チームナーシングでチーム力を活かした看護に取り組む
- 地域の皆様に信頼される医療を提供する
- 一人ひとりの患者様に丁寧に向き合える落ち着いた環境の

### 型C: 経験とのフィット

対象: 40代〜 × 経験1〜2年

```
{特色}当院において、これまでの看護師としてのご経験を活かしていただけるのではないかと考え、ご連絡しました。
```

特色バリエーション:
- 急性期と地域包括ケアの両病棟で多様な看護を提供する
- 固定チームナーシングでスタッフ同士が支え合いながら看護に取り組む
- 地域に根差した医療を提供する

### 型D: 経験ある前提で評価

対象: 40代〜 × 経験未入力
注意: 教育体制の訴求は上から目線になるため使わない

就業中:
```
現在も看護師として臨床の現場で培われている経験やスキルは、{特色}当院で活かしていただけると考えています。
```

離職中:
```
看護師としてこれまで培われてきた経験やスキルは、{特色}当院で活かしていただける場面が多いと考えています。
```

特色バリエーション:
- 急性期から地域包括ケアまで幅広い患者様に対応する
- チーム全体で患者様を支える固定チームナーシングを実践する
- 多診療科の症例に対応し看護の幅を広げられる

### 型E: ポテンシャル+教育体制

対象: 20〜30代 × 経験1〜2年

```
看護師として臨床の現場で積んでこられたご経験は、当院でさらに広げていただけます。中途入職の方にも専任担当者が約3ヶ月間マンツーマンでサポートする体制が整っており、その日の疑問をその日のうちに解消できる環境です。
```

### 型F: 教育体制+成長環境

対象: 20〜30代 × 経験未入力

就業中:
```
現在も看護師として臨床の現場で培われているスキルは、当院でも十分に活かしていただけます。中途入職の方にも約3ヶ月間マンツーマンでサポートする体制が整っており、急性期から地域包括ケアまで幅広い看護経験を積める環境です。
```

離職中:

## AI生成パス用ガイド

経歴・自己PRに実質的な情報がある候補者に使用する。

### 生成手順

1. 候補者の経歴・自己PRから最も強い接点を1つ選ぶ
   - 経験年数が4年以上の候補者には、パーソナライズ文中に具体的な経験年数を含めること（例: 「10年以上の臨床経験」「5年にわたる〜」）
2. 上記「当院の特色」から対応する強みを選ぶ
3. **候補者のライフステージ・状況に応じた訴求ポイント**を組み合わせる（下記参照）
4. 接点を100文字程度で表現する

### ライフステージ別の訴求ポイント

経歴の接点に加え、**プロフィールに明記されている情報のみ**に基づいて以下の要素を組み合わせる。

**絶対ルール: 確証のない属性（子育て中、家庭の事情、転職理由等）にはパーソナライズ文で一切言及しない。**
託児所・時短勤務等の制度はテンプレート本文に記載済みのため、パーソナライズ文で重ねて訴求する必要はない。

| プロフィールに明記されている情報 | 刺さりやすい訴求 | 使える素材 |
|-------------------------------|---------------|----------|
| 大規模急性期病院の勤務歴 | 忙しすぎない急性期 | 救急搬送1日6〜7件、質問できる雰囲気、その日の疑問をその日のうちに解消 |
| ブランクありと明記 | 放置しない教育体制 | 3ヶ月マンツーマンサポート、新卒離職率10年連続0%の風土が中途にも適用 |
| 経験年数が浅い（1〜2年） | 成長できる環境 | 専任担当者のサポート、eラーニング、院内研修 |
| 自己PRで長く働きたいと明記 | 長期キャリアの安定基盤 | 徳洲会グループの安定性、賞与3ヶ月、グループ内異動で長期キャリア継続 |
| 自己PRでチーム医療に言及 | チームの風土 | 固定チームナーシング、お互い様の文化 |
| こだわり条件で給与を重視 | 具体的な数字 | 経験3年で年収495万、夜勤手当15,000円/回 |

### NGパターン

- 運転免許など弱い接点で文を構成する
- 会社情報を羅列するだけで接点になっていない
- 地理的要素のみに頼る（テンプレートに記載済み）
- 候補者の希望職種が看護師以外でも、看護師としての接点で書く（希望職種には言及しない）
- ライフステージを決めつける表現（「子育て中のあなたに」「お子様がいらっしゃると思いますが」等）
- 制度の羅列だけで接点になっていない（経歴との接点が先、制度は添え物）
