# パーソナライズ レシピ: an訪問看護ステーション

## 当ステーションの特色（パーソナライズで使える素材）

- 横浜市南区を拠点に地域密着の訪問看護・訪問リハビリ
- 訪問エリア: 南区全域、磯子区、中区、保土ヶ谷区、西区、港南区の一部
- 看護師 + リハビリ職（PT/OT）が在籍、多職種連携で利用者様を支える
- ターミナルケア・精神障害・認知症ケアにも対応
- ワークライフバランス重視（9〜17時・7時間勤務、残業ほぼなし、年間休日124日）
- 入社初日から見学同行スタート、1ヶ月程度で独り立ち

## 共通ルール

- 書き出し: テンプレート冒頭に「ご経歴を拝見し」があるため「拝見し」以外で開始する
- 文字数: 初回 約100文字 / 再送 120〜150文字
- トーン: 丁寧で誠実。堅くなりすぎない自然な敬語
- 地理情報（勤務地・訪問範囲）はテンプレートに記載済みのため言及しない
- 年齢・世代への言及禁止（「〇代」「若手」「ベテラン」等）
- 経験年数の記載ルール:
  - 1〜2年: 書かない
  - 3年: 30代前半以下なら書く、30代後半以上なら書かない
  - 4年以上: 書く
- 「感銘を受ける」は経験そのものに使用禁止。代替:「魅力を感じました」「注目しました」
- こだわり条件は参考程度。資格・経験で接点が作れるなら言及不要
- 上から目線の排除（「フォローします」「安心してください」「基礎を固めてこられた」）
- 送り手の感情主体（「ご一緒したい」「想いを強く感じ」）は避け、相手へのオファー主体で
- ですます調で統一。体言止めや「〜とのこと、」で文を切らない
- 憶測を書かない（「〜されたいのですね」「長く続けているはず」）
- 不要な前置きを書かない（「現在離職中とのこと」「新たな環境でとお考えの〜」）
- 居住地に言及する場合は広域表現に留める（「〇〇区」→「横浜市内」「神奈川県内」）
- 堅苦しい敬語禁止（「拝察いたします」「敬意を表します」等）

## 教育体制（当ステーション固有 — 他社の制度を混ぜないこと）

- 入社初日から見学同行スタート、1ヶ月程度で独り立ち
- 社内研修制度あり
- 資格取得支援制度あり

## 型はめパターン（看護師用）

情報が少ない候補者（経歴・自己PRなし）に使用する。
特色バリエーションは候補者ごとにローテーションし、同じ文の連続を防ぐ。

### 型A: 豊富な経験への期待

対象: 経験10年+ / 40代〜×経験6年+

```
10年以上にわたり看護の現場でご活躍されてきたご経歴を、大変心強く拝見しました。豊富な臨床経験で培われた確かな技術とアセスメント力は、{特色}当ステーションにおいて、大きな力になると考えています。
```

特色バリエーション:
- 利用者様一人ひとりに寄り添うケアを大切にする
- 看護師とリハビリ職が連携して利用者様を支える
- ターミナルケアや認知症ケアにも対応する

### 型B1: 確かな経験×特色

対象: 経験6〜9年

```
看護師として{N}年にわたり臨床の現場で培われた確かな経験やスキルは、{特色}当ステーションにおいて、大きな力になると考えています。
```

特色バリエーション:
- 看護師とリハビリ職が連携しながら訪問看護に取り組む
- 地域に根差した訪問看護で利用者様の生活に寄り添う
- ターミナルケアから日常支援まで幅広く対応する

### 型B2: 経験×特色

対象: 経験3〜5年

```
看護師として{N}年にわたり現場で培われたスキルは、{特色}当ステーションにおいて活かしていただけると考えています。
```

特色バリエーション:
- 多職種が連携しチームで利用者様を支える
- 地域に根差したケアを提供する
- 利用者様一人ひとりの暮らしに寄り添う

### 型C: 経験とのフィット

対象: 40代〜 × 経験1〜2年

```

```

特色バリエーション:
- 利用者様一人ひとりに寄り添うケアを大切にする

## 資格修飾（複数資格がある場合、型の冒頭を差し替え）

型はめパターンの書き出し部分を以下に差し替える。後半（特色部分）はそのまま使う。

### 看護師 + 保健師

```
看護師と保健師の両資格をお持ちで、保健師として培われた予防的視点やアセスメント力は、在宅で利用者様の生活全体を支える訪問看護において大きな強みになると考えています。
```

### 看護師 + ケアマネージャー

```
看護師の資格に加え、ケアマネージャーの資格もお持ちとのこと、医療と介護の両面から利用者様を支える視点は、多職種と連携しながら訪問看護に取り組む当ステーションで大きな力になると考えています。
```

### 看護師 + 認定看護師等の専門資格

```
{資格名}の資格をお持ちで、その専門性は{関連する特色}当ステーションにおいて、大きな力になると考えています。
```

### PT/OT + 他のリハビリ資格

```
{職種名}に加え、{他資格名}の資格もお持ちとのこと、複数の視点からアプローチできる力は、多職種が連携する当ステーションで大きな強みになると考えています。
```

## AI生成パス用ガイド

経歴・自己PRに実質的な情報がある候補者に使用する。

### 生成手順

1. 候補者の経歴・自己PRから最も強い接点を1つ選ぶ
   - 経験年数が4年以上の候補者には、パーソナライズ文中に具体的な経験年数を含めること（例: 「10年以上の臨床経験」「5年にわたる〜」）
2. 上記「当ステーションの特色」から対応する強みを選ぶ
3. 接点を100文字程度で表現する

### 経歴別の接点対応表（看護師）

| 候補者の経験 | 対応するステーション特色 | 接点の方向性 |
|------------|----------------------|------------|
| 認知症ケア | 認知症ケアに対応 | 「認知症ケアのご経験は、当ステーションでの訪問看護で大きな力になる」 |
| ターミナルケア | ターミナルケア対応 | 「看取りに向き合ってこられた経験は、当ステーションが大切にしている姿勢と重なる」 |
| 病棟看護（急性期等） | 多職種連携 | 「病棟でのチーム連携のご経験は、看護師とリハビリ職が連携する当ステーションで活かせる」 |
| 訪問看護経験 | 地域密着 | 「訪問看護で培われた対話力は、利用者様との関係性作りにおいて大きな強みになる」 |
| 精神科看護 | 精神障害ケアに対応 | 「精神科での看護経験は、多様な利用者様に寄り添う当ステーションで」 |

### 経歴別の接点対応表（リハビリ職）

| 候補者の経験 | 対応するステーション特色 | 接点の方向性 |
|------------|----------------------|------------|
| 脳血管リハビリ | 脳血管リハビリ対応 | 「脳血管領域でのリハビリ経験は、当ステーションの利用者様のケアに」 |
| 運動器リハビリ | 運動器リハビリ対応 | 「運動器リハビリの経験を、在宅でのリハビリに活かしていただける」 |
| 訪問リハビリ経験 | 地域密着 | 「訪問リハビリの経験を活かし、即戦力として」 |
| 病院リハビリ | 多職種連携 | 「病院で培われた臨床経験を、訪問の現場で活かしていただける」 |
| 精神科リハビリ | 精神障害ケア対応 | 「精神科リハビリの経験は、多様な利用者様に寄り添う当ステーションで」 |
| 難病リハビリ | 難病リハビリ対応 | 「難病リハビリのご経験は、幅広い利用者様に対応する当ステーションで」 |

### NGパターン

- 運転免許など弱い接点で文を構成する
- 会社情報を羅列するだけで接点になっていない
- 地理的要素のみに頼る
- 候補者の希望職種と異なる資格の接点で書く
