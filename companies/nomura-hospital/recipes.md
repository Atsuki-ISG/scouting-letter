# パーソナライズ レシピ: 野村病院

## 看護師

### 当院の特色（パーソナライズで使える素材）

- 全147床の精神科専門病院（精神科・神経内科・一般内科）
- 患者様との対話と個性を大切にする精神科医療
- 精神科未経験・ブランクOKで多くのスタッフが活躍中
- 管理職候補は月給37万円以上、キャリアアップの道が明確
- 車通勤可（駐車場有）・寮完備で遠方からの就業もサポート
- 入職前の院内見学・業務体験でミスマッチを防止

### 共通ルール

- 書き出し: テンプレート冒頭に「ご経歴を拝見し」があるため「拝見し」以外で開始する
- 文字数: 初回 約100文字 / 再送 120〜150文字
- 地理情報（勤務地）はテンプレートに記載済みのため言及しない
- 年齢・世代への言及禁止（「〇代」「若手」「ベテラン」等）
- 経験年数の記載ルール:
  - 1〜2年: 書かない
  - 3年: 30代前半以下なら書く、30代後半以上なら書かない
  - 4年以上: 書く
- 「感銘を受ける」は経験そのものに使用禁止。代替:「心強く拝見」「注目しました」
- こだわり条件は参考程度。資格・経験で接点が作れるなら言及不要
- 上から目線の排除（「フォローします」「安心してください」「基礎を固めてこられた」）
- 送り手の感情主体（「ご一緒したい」「想いを強く感じ」）は避け、相手へのオファー主体で
- ですます調で統一。体言止めや「〜とのこと、」で文を切らない
- 憶測を書かない（「〜されたいのですね」「長く続けているはず」）
- 不要な前置きを書かない（「現在離職中とのこと」「新たな環境でとお考えの〜」）
- 居住地に言及する場合は広域表現に留める（「〇〇市」→「山口県内」）
- 堅苦しい敬語禁止（「拝察いたします」「敬意を表します」等）

### 教育体制（当院固有 — 他社の制度を混ぜないこと）

- 現場のスタッフと協力しながら業務を覚えられる体制
- 入職前の院内見学・説明会・簡易的な業務体験
- ※ eラーニングや研修費補助等の制度は記載なし。他社の制度を混ぜないこと

### 型はめパターン

情報が少ない候補者（経歴・自己PRなし）に使用する。
特色バリエーションは候補者ごとにローテーションし、同じ文の連続を防ぐ。

### 型A: 豊富な経験への期待

対象: 経験10年+ / 40代〜×経験6年+

```
10年以上にわたり看護の現場でご活躍されてきたご経歴を、大変心強く拝見しました。豊富な臨床経験で培われた確かな技術とアセスメント力は、{特色}当院において、大きな力になると考えております。
```

特色バリエーション:
- 患者様との対話を大切にする精神科医療を提供する
- 147床の病棟で多様な症状の患者様に対応する
- 地域に根差した精神科医療を実践する

### 型B1: 確かな経験×特色

対象: 経験6〜9年

```
看護師として{N}年にわたり臨床の現場で培われた確かな経験やスキルは、{特色}当院において、大きな力になると考えております。
```

特色バリエーション:
- 患者様一人ひとりの症状に合わせたケアを提供する
- 対話を通じた精神科看護を実践する
- 地域の精神科医療を支える

### 型B2: 経験×特色

対象: 経験3〜5年

```
看護師として{N}年にわたり現場で培われたスキルは、{特色}当院において活かしていただけると考えております。
```

特色バリエーション:
- 患者様との対話を重視した精神科看護に取り組む
- 多様な症状の患者様にきめ細かく対応する
- 地域に根差した精神科医療を提供する

### 型C: 経験とのフィット

対象: 40代〜 × 経験1〜2年

```
{特色}当院において、これまでの看護師としてのご経験を活かしていただけるのではないかと考え、ご連絡いたしました。
```

特色バリエーション:
- 患者様一人ひとりに寄り添った精神科医療を実践する
- 対話を通じたケアを大切にする
- 地域の精神科医療を支えている

### 型D: 経験ある前提で評価

対象: 40代〜 × 経験未入力
注意: 教育体制の訴求は上から目線になるため使わない

就業中:
```
現在も看護師として臨床の現場で培われている経験やスキルは、{特色}当院で活かしていただけると考えております。
```

離職中:
```
看護師としてこれまで培われてきた経験やスキルは、{特色}当院で活かしていただける場面が多いと考えております。
```

特色バリエーション:
- 患者様との対話を大切にする精神科看護に取り組む
- 多様な症状の患者様に寄り添ったケアを提供する
- 地域に根差した精神科医療を実践する

### 型E: ポテンシャル+成長環境

対象: 20〜30代 × 経験1〜2年

```
看護師として臨床の現場で積んでこられたご経験は、精神科という新たなフィールドでさらに広げていただけます。当院では精神科未経験から活躍しているスタッフも多く、チームでサポートし合える環境です。
```

### 型F: 成長環境+チーム体制

対象: 20〜30代 × 経験未入力

就業中:
```
現在も看護師として臨床の現場で培われているスキルは、精神科看護でも十分に活かしていただけます。当院では精神科未経験から活躍しているスタッフも多く、キャリアの幅を広げられる環境が整っております。
```

離職中:
```
当院では精神科未経験から活躍しているスタッフも多く、現場のスタッフと協力しながら取り組める環境です。
```

### 型G: 成長環境メイン

対象: 在学中

```
これからのキャリアを検討されているとのこと、当院では精神科未経験の方も多く活躍しており、現場のスタッフと協力しながら精神科看護に取り組める環境が整っております。
```

### AI生成パス用ガイド

経歴・自己PRに実質的な情報がある候補者に使用する。

#### 生成手順

1. 候補者の経歴・自己PRから最も強い接点を1つ選ぶ
   - 経験年数が4年以上の候補者には、パーソナライズ文中に具体的な経験年数を含めること（例: 「10年以上の臨床経験」「5年にわたる〜」）
2. 上記「当院の特色」から対応する強みを選ぶ
3. 接点を100文字程度で表現する

#### 経歴別の接点対応表

| 候補者の経験 | 対応する当院の特色 | 接点の方向性 |
|------------|------------------|------------|
| 精神科看護 | 147床の精神科専門病院 | 「精神科看護のご経験は、当院の病棟で即座に活かしていただけます」 |
| 病棟看護（急性期・慢性期等） | 精神科未経験OK | 「病棟看護で培われたアセスメント力や観察力は、精神科看護でも十分に発揮いただけます」 |
| 認知症ケア | 対話を大切にする精神科看護 | 「認知症ケアで培われたコミュニケーション力は、対話を大切にする当院の精神科看護で大きな強みになります」 |
| 訪問看護・在宅医療 | 患者様に寄り添うケア | 「在宅で患者様に寄り添ってこられたご経験は、精神科看護の核となる対話力として活かしていただけます」 |
| リーダー・主任・管理職 | 管理職候補で月給37万以上 | 「マネジメントのご経験を、当院の病棟運営においてぜひ発揮いただきたいと考えております」 |
| 複数診療科の経験 | 精神科+内科の複合診療 | 「幅広い臨床経験は、精神科に加え内科も併設する当院で存分に活かしていただけます」 |
| コミュニケーション重視の姿勢 | 対話と個性を大切にする方針 | 「患者様との対話を重視されるお考えは、当院の理念と深く重なります」 |

### NGパターン

- 運転免許など弱い接点で文を構成する
- 会社情報を羅列するだけで接点になっていない
- 地理的要素のみに頼る
- 候補者の希望職種が看護師以外でも、看護師としての接点で書く（希望職種には言及しない）
- 他社の教育制度（eラーニング、研修費補助等）を当院の特色として記載する

## 管理栄養士/栄養士（契約職員） レシピ

### 当院の特色（パーソナライズで使える素材）

- 精神科病院での栄養管理（心と体の両面からのケア）
- 未経験OK — 管理栄養士資格があれば業務経験不問
- 残業ほぼなし、夜勤なし（日勤のみ）
- 賞与年2回・計3ヶ月分
- マイカー通勤可（通勤手当あり）

### 共通ルール

- 書き出し: テンプレート冒頭に「ご経歴を拝見し」があるため「拝見し」以外で開始する
- 文字数: 初回 約100文字 / 再送 120〜150文字
- 地理情報（勤務地）はテンプレートに記載済みのため言及しない
- 年齢・世代への言及禁止（「〇代」「若手」「ベテラン」等）
- 産休代替・契約期間には触れない
- 看護師求人の情報を混ぜない
- 上から目線の排除（「フォローします」「安心してください」「基礎を固めてこられた」）
- ですます調で統一。体言止めや「〜とのこと、」で文を切らない
- 憶測を書かない
- 不要な前置きを書かない
- 居住地に言及する場合は広域表現に留める
- 堅苦しい敬語禁止

### 型はめパターン

情報が少ない候補者（経歴・自己PRなし）に使用する。

### 型A: 病院・施設での栄養管理経験あり

対象: 経歴に病院・施設での栄養管理経験がある

```
病院での栄養管理業務に携わってこられたご経験は、精神科医療を提供する当院において大いに活かしていただけると考えております。{特色}
```

特色バリエーション:
- 残業もほぼなく、安定した働き方で長くご活躍いただける環境です。
- 日勤のみの勤務で、生活リズムを保ちながらご活躍いただけます。

### 型B: 栄養管理の実務経験あり（病院以外）

対象: 給食センター・保育園・介護施設等での経験

```
管理栄養士として培ってこられた栄養管理のスキルは、{特色}当院で活かしていただけると考えております。
```

特色バリエーション:
- 精神科医療を通じて患者様の心と体を支える
- 栄養管理・材料管理・衛生管理を幅広く担う

### 型C: 経験未入力・経験浅め

対象: 経歴の記載がない / 経験が浅い

就業中:
```
管理栄養士として現場で培われているスキルは、当院の栄養管理業務で活かしていただけると考えております。残業もほぼなく、日勤のみの安定した環境でご活躍いただけます。
```

離職中:
```
管理栄養士の資格を活かせるポジションのご案内です。当院では業務経験を問わず、残業もほぼなく日勤のみの環境で、無理なく取り組んでいただけます。
```
